## 背景  
数据库规划  
  
```  
1  master - standby  
5  segment(s) - segment mirror(s)  
```

master节点配置(cpu 8核, mem 16G, network 1GB, disk 1*250G)  
  
segments配置


交换KEY，master使用gp管理用户(digoal)访问所有的segment不需要输入密码，master pub拷贝到所有的segment authorized_keys  
  
```  
$ gpssh-exkeys -f ./host  
```  
  
安装软件到segment hosts  
  
```  
$gpseginstall -f ./host -u digoal  
```
