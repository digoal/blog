进入老的主库数据目录，发现多了两个文件  
  
```  
cd /disk1/digoal/gpdata/gpseg-1  
-rw-r--r-- 1 digoal users     0 Dec 22 16:51 promote  
-rw-r--r-- 1 digoal users     0 Dec 22 16:51 recovery.conf  
```  
  
promote代表要激活它，recovery.conf没有用。  
  
把这两个文件删掉。
