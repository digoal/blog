2、写一个SHELL脚本，自动拷贝依赖的动态库。  
  
```  
vi test.sh  
  
deplist=$( ldd $1 | awk '{if (match($3,"/")){ print $3}}' )  
cp -L -n $deplist $2   
```  
  
```  
-L 表示拷贝文件（如果是软链，会拷贝到最终的文件）。  
-n 表示不覆盖已有文件。  
```  
  
3、创建目标目录。  
  
```  
mkdir /tmp/lib  
```  
  
4、将postgres依赖的动态库拷贝到目标目录  
  
```  
. ./test.sh /usr/pgsql-10/bin/postgres /tmp/lib/  
```
