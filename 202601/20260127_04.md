## 背景  
今天, 我们来用 AI 深度分析某大厂的财报.  
  
以某上市公司为例, 进入公司官网, 下载一份财报.  
  
https://ir.tesla.com/  
  
我下载的是这个: 2025 Q3 财报.  
  
然后把 PDF 丢给 AI, 从以下提示词开始发问:  
```  
“你现在是一名资深财务分析师。请阅读这份财报，并为我总结：  
这家公司本年度最核心的收入来源是什么？  
与去年相比，毛利率发生了什么变化？原因是什么？  
财报中提到的‘主要经营风险’有哪些？  
经营活动产生的现金流是否能够覆盖其净利润？”  
```  
  
下面我们一起来学习如何用 AI 从财报中获取对投资有价值的内容, 有看不懂的地方不要方张, 后面我会逐个展开继续提问 AI. 问到搞明白为止.

作为资深财务分析师，我为您总结了特斯拉（Tesla）2025年第三季度财报的核心财务状况与经营风险：  
  
### 1. 本年度最核心的收入来源  
  
特斯拉最核心的收入来源依然是**汽车业务（Automotive）** 。  
  
* **收入规模**：2025年第三季度，汽车业务总收入达到**212.05亿美元**，占该季度总营收（280.95亿美元）的约**75.5%** 。  
* **增长情况**：该项收入同比去年增长了**6%** ，主要由车辆交付量的增加驱动 。  
  
  
  
### 2. 毛利率的变化及原因  
  
与去年同期相比，**GAAP毛利率有所下降**，但呈现出季度环比改善的趋势。  
  
* **数值变化**：2025年Q3的总GAAP毛利率为**18.0%** ，相较于2024年Q3的19.8%下降了**185个基点（bp）** 。  
* **下降原因**：  
    * **成本增加**：由于某些车型的固定成本吸收能力下降、关税增加以及销售结构（mix）的变化，导致**单车平均成本（Average cost per vehicle）上升** 。  
    * **收入减少**：监管信贷（Regulatory Credit）收入以及FSD（全自动驾驶）的一次性收入确认均同比减少 。  
* **抵消因素**：原材料成本的下降，以及能源生成与储能业务、服务及其他业务毛利的增长部分抵消了上述负面影响 。  
  
  
  
### 3. 财报中提到的“主要经营风险”  
  
财报明确指出了以下几项影响未来发展的挑战与不确定性：  
  
* **宏观经济与政策风险**：全球贸易环境、关税政策以及财政政策的变化（如美国电动车税务抵免到期）带来了近期不确定性 。  
* **执行与生产风险**：实际业绩取决于自动驾驶技术的推进速度、新工厂的生产提升效率以及宏观经济环境 。  
* **供应链风险**：汽车和能源供应链的成本结构容易受到外部环境冲击 。  
* **监管审批风险**：如FSD（监督版）在欧洲、中国等地区的推出仍取决于监管部门的审批 。  
  
  
  
### 4. 经营活动现金流是否覆盖净利润  
  
**是的，经营活动产生的现金流大幅覆盖了其净利润。**  
  
* **净利润（GAAP）** ：2025年Q3为**13.89亿美元** 。  
* **经营活动现金流**：2025年Q3为**62.38亿美元** 。  
* **分析结论**：经营现金流约为净利润的**4.5倍**。这表明公司拥有极强的获现能力，利润的质量较高。即使在扣除22.48亿美元的资本支出后，公司仍产生了近40亿美元（39.90亿美元）的自由现金流 。
