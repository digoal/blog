Unix/Linux平台下如何查看MAC\WWN地址

2011-01-11 19:40:13   查看原文>>

好东西,必须要[转]



归纳了一下，在UNIX/LINUX平台下查看MAC和WWN地址的方法

查看MAC地址

hp-ux

#lanscan

 

tru64 unix

#netstat -ia

 

aix

#lsdev -Cc if -S a

#lscfg -vpl entX

#netstat -ia

 

linux

#ifconfig -a

 

solaris

#ifconfig -a

 

查看WWN地址

aix

#lscfg -vpl fcs0

#lscfg -vpl fcs1

 

solaris

#prtpicl Cc scsi-fcp Cv | grep Ci wwn (对于Qlogic HBA)

# luxadm Ce port

# luxadm Ce dump_map <physical_path_listed_on_above_cli>

 

hp-ux

#ioscan -fnC fc

#fcmsutil /dev/fcd0

 

redhat linux

#view /proc/scsi/

kernel 2.6

WWPN:

cat /sys/class/fc_host/host*/port_name

WWNN:

cat /sys/class/fc_host/host*/node_name

 

tru64 unix

#fgrep wwn /var/adm/messages
