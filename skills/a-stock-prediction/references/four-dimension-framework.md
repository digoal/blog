# 四维分析框架(详细规则)

> 这是 SKILL.md 步骤 3 的展开。每个维度都有"打分细则、典型证据、常见陷阱"三部分。
> 写预测前必须先 Read 本文件,确保每条打分都有据可依。

## 总览

| 维度 | 权重 | 核心问题 | 打分范围 |
|---|---|---|---|
| 大盘环境 | 30% | 大盘是给个股加 buff 还是减 buff? | -2 ~ +2 |
| 资金行为 | 30% | 大钱是流入还是流出?谁在买/卖? | -2 ~ +2 |
| 技术面 | 25% | 价格在哪个关键位?趋势是否健康? | -2 ~ +2 |
| 消息面 | 15% | 有没有"非交易型"的事件冲击? | -2 ~ +2 |

加权求和后,最终分映射到方向与概率(见 SKILL.md 步骤 4)。

## 维度 1:大盘环境(权重 30%)

A股是高度系统性的市场。**强势市场里弱股也能涨,弱势市场里强股也会跌。**

### 打分细则(打分 = 满足项数对应的分值)

| 打分 | 触发条件(满足 4 项以上) |
|---|---|
| **+2** | 四大指数全部站上 5 日均线 + 涨停股 ≥ 80 家 + 跌停股 < 10 家 + 主线连板高度 ≥ 3 板 + 北向净流入 ≥ 50 亿 + 涨停股次日红盘率 > 60% |
| **+1** | 上述条件满足 3-4 项 |
| **0** | 指数震荡 / 涨跌家数平衡 / 主线断板 / 北向小幅净流出 |
| **-1** | 指数跌破 10 日均线 + 跌停 > 30 家 + 主线无连板 + 北向净流出 > 30 亿 |
| **-2** | 指数全线跌破年线 + 千股跌停 + 北向恐慌性卖出 > 100 亿 + 涨停股次日红盘率 < 30% |

### 关键指标速查表

| 指标 | 强势阈值 | 弱势阈值 | 数据来源关键词 |
|---|---|---|---|
| 上证指数 5 日线站位 | 站上 | 跌破 | "上证指数 收盘 涨跌" |
| 涨停股家数 | > 80 家 | < 30 家 | "今日 涨停股 数量" |
| 跌停股家数 | < 10 家 | > 50 家 | "今日 跌停股 数量" |
| 主线连板高度 | ≥ 3 板 | 0 连板 | "今日 高度板 连板梯队" |
| 北向资金净流向 | 净流入 > 50 亿 | 净流出 > 50 亿 | "北向资金 今日 净流入" |
| 涨跌家数比 | 涨 > 3000 家 | 跌 > 3000 家 | "今日 涨跌家数 A股" |
| 两市成交额 | 万亿以上(活跃) | < 6000 亿(萎缩) | "今日 沪深两市 成交额" |

### 典型证据来源

- 东方财富 → "盘面"页
- 同花顺 i 问财
- 集思录 → 涨跌停统计
- Wind / 通达信涨停板情报
- 财联社 - 涨停板池

### 常见陷阱

1. **看错了指数权重**:上证指数被大金融拖累,但创业板/中证 2000 才是题材股的真实环境
2. **忽视风格切换**:大盘股涨、小盘股跌(或反之)是常见结构性行情,不能用单一指数判断
3. **把"涨停股多"等同于"赚钱效应好"**:要看"涨停股次日红盘率",这才是真正的连续赚钱信号
4. **北向资金的滞后性**:北向数据 15 分钟延迟,且节假日休市,不能视为实时领先指标

## 维度 2:资金行为(权重 30%)

资金是最不会说谎的指标。看谁在买、买了多少、买的是谁。

### 打分细则

| 打分 | 触发条件 |
|---|---|
| **+2** | 龙虎榜上知名游资席位净买入 > 5000 万 + 主力资金净流入 > 流通市值 1% + 北向加仓 + 融资余额放大 |
| **+1** | 龙虎榜净买入或机构席位净买入 + 主力净流入或大单买入比 > 1.5 |
| **0** | 无龙虎榜 + 主力资金小幅流入/流出 + 北向无明显动作 |
| **-1** | 主力净流出 > 流通市值 1% / 龙虎榜净卖出 / 北向减持 |
| **-2** | 连续 3 日主力净流出 + 龙虎榜知名游资砸盘 + 北向恐慌性卖出 + 融资净偿还放量 |

### 关键指标速查表

| 信号 | 强度判断 | 数据来源关键词 |
|---|---|---|
| **龙虎榜** | 知名游资席位 + 净买入 > 5000 万 = 强;机构净买入 = 中长线信号;一日游游资 = 谨慎 | "{代码} 龙虎榜 今日" |
| **主力资金净流入** | 净流入 > 流通市值 1% = 强势;净流出 > 1% = 弱势 | "{代码} 主力资金 流入" |
| **北向资金** | 持股比例增加 = 加分;连续 3 日减持 = 减分 | "{代码} 北向 持股比例" |
| **融资余额** | 融资净买入放大 = 杠杆资金看多;融资净偿还 = 看空 | "{代码} 融资余额 变化" |
| **大单买卖比** | 买入大单 / 卖出大单 > 1.5 = 主力承接 | "{代码} 大单 净买入" |
| **涨停封单** | 封单 > 流通盘 10% = 次日大概率溢价 | "{代码} 涨停 封单金额" |

### 知名游资席位识别(部分代表性席位)

- **中国银河 绍兴**:章盟主(题材龙头打板)
- **国泰君安 上海打浦路**:章盟主第二席位
- **华鑫证券 上海分公司**:陈小群
- **方新侠**:深圳新城路营业部
- **东方财富 拉萨**(多个):游资聚集地
- **机构席位**:出现"机构专用"字样

详见 `references/terminology.md` 中"游资席位"词条。

### 风险信号(出现一个就大幅减分)

- 龙虎榜出现"接力游资砸盘"(前一日买入的游资当日卖出)
- 主力连续 3 日净流出且阴跌(主力在出货)
- 北向连续 3 日大幅减持(外资看空信号)
- 融资余额单日骤降 > 5%(杠杆资金恐慌性减仓)
- "机构专用"席位在龙虎榜上卖出(机构出货,杀伤力大)

### 常见陷阱

1. **把"成交量大"等同于"主力买入"**:成交量大可能是机构出货,要看大单方向
2. **忽视"对倒"风险**:机构+游资同时买入卖出 = 可能是自买自卖制造活跃假象
3. **北向资金不等于聪明钱**:北向数据滞后,且包含部分内资伪装的"假外资"
4. **融资余额放大≠看多**:也可能是大资金借钱买入后再出货,反而是顶部信号
5. **龙虎榜买一独大但卖一散户**:看似强势,但如果买一是"一日游"游资,次日很可能砸盘

## 维度 3:技术面(权重 25%)

技术面不预测未来,只描述当下的供需状态。**A股技术面比美股更有效**,因为散户占比高、技术派多。

### 打分细则

| 打分 | 触发条件 |
|---|---|
| **+2** | 放量突破前期高点 + 多头排列(5>10>20>60 日均线) + 量比 > 2 + 收阳实体大 |
| **+1** | 站稳 5 日均线 + 量价配合 + 形态偏多(上升通道、旗形整理) |
| **0** | 区间震荡 + 均线缠绕 + 无明显方向 |
| **-1** | 跌破 10 日均线 + 缩量阴跌 + 形态偏空(下降通道、M 顶) |
| **-2** | 跌破年线 + 量价齐跌 + 跌破前期低点 + 均线空头排列 |

### 关键观察点

**K线与量价(权重最高):**
- 放量上涨收阳实体大 → 强势延续(+5 分项)
- 放量上涨收长上影 → 见顶警惕(-3 分项)
- 缩量回调到 5/10 日均线 → 蓄势待发(+3 分项)
- 放量阴跌 → 主力出货(-5 分项)
- 缩量阴跌 → 弱势但未到底(-2 分项)
- 量比 = 当日成交量 / 近 5 日平均成交量,> 2 表示放量,< 0.7 表示缩量

**均线系统:**
- 多头排列(5 > 10 > 20 > 60)+ 站稳 5 日均线 = 趋势良好
- 跌破 10 日均线 = 短线走弱
- 跌破 20 日均线 = 中线趋势减弱
- 跌破 60 日均线 = 中长线趋势破坏
- 跌破年线(250 日均线)= **趋势完全破坏,放弃**

**关键位置:**
- 放量突破前期高点 = 多头确认(+5 分项)
- 放量跌破前期低点 = 空头确认(直接放弃)
- 接近大级别压力位(前期顶部、整数关口) = 资金会兑现,减分
- 接近大级别支撑位(前期底部、年线、密集成交区) = 有承接,加分

**形态识别:**
- 利多形态:上升通道、旗形整理、矩形整理、底背离、W 底
- 利空形态:下降通道、M 顶、头肩顶、顶背离、岛形反转

### 数据来源关键词

```
{代码} K线 5日均线 10日均线 20日均线
{代码} 今日 成交量 量比
{代码} 近 5 日 最高 最低 收盘
{代码} 技术分析 支撑 压力
```

### 常见陷阱

1. **滞后性**:技术面是结果不是原因,大资金已经知道才出现技术信号,散户看到时往往是末班车
2. **假突破**:小范围突破前高 +1% 又快速跌回,是"诱多"出货
3. **缩量回调要分情况**:在上升通道里缩量回调是健康,在顶部缩量回调是阴跌前奏
4. **看错均线周期**:超短线看 3 / 5 日,短线看 10 / 20 日,中线看 60 / 120 日,长线看 250 日(年线)
5. **MACD / KDJ 等指标失效**:在震荡市频繁假信号,只在趋势明确时有效

## 维度 4:消息面(权重 15%)

消息是催化剂,能放大趋势但不能扭转趋势。**A股是政策市,消息面权重虽小但杀伤力大。**

### 打分细则

| 打分 | 触发条件 |
|---|---|
| **+2** | 业绩大幅超预期(单季净利同比 > +50%)+ 国家级行业政策利好 / 重大订单(单笔 > 公司年营收 10%) / 重组并购落地 |
| **+1** | 业绩小幅超预期 / 部委级行业政策 / 大订单签订 / 股份回购公告 |
| **0** | 无重大消息 / 消息已被市场消化 |
| **-1** | 业绩小幅低于预期 / 行业政策利空 / 高管离职 / 商誉减值预警 |
| **-2** | 业绩暴雷(单季亏损或同比 < -50%)/ 限售解禁(大股东) / 证监会立案调查 / 重大产品召回 / 大股东减持公告 |

### 利好消息分类

**强催化(+2):**
- 业绩超预期(实际 EPS 比券商一致预期高 > 10%)
- 国家级行业政策利好(如"低空经济发展规划"发布)
- 重大订单 / 中标公告(单笔 > 年营收 10%)
- 并购重组(注意核实真实性,避免炒概念)
- 股份回购(回购金额 > 5% 市值)

**中性偏多(+1):**
- 行业政策利好(部委级)
- 大订单签订
- 引入战略投资者
- 板块龙头异动带动跟风

### 利空消息分类(任一出现建议直接放弃)

**严重利空(-2):**
- 业绩暴雷或预亏
- 限售股次日解禁(特别是大股东解禁,流通盘大幅扩大)
- 大股东减持公告
- 证监会立案调查
- 行业政策利空(如"AI 监管收紧")
- 商誉减值预警(年报季)

**警惕信号(-1):**
- 业绩低于市场预期
- 高管离职(尤其是 CFO 离职)
- 关联交易争议
- 大额质押风险

### 必查事项

- 次日是否有重要公告日(年报、季报、股东大会、可转债转股日)
- 是否有除权除息(股价会自动调整,影响 K 线判断)
- 是否有股东大会重大事项表决

### 数据来源关键词

```
{代码} 最新公告 巨潮资讯
{代码} 业绩预告 {季度}
{代码} 限售解禁 时间表
{代码} 大股东 减持 公告
{行业} 最新政策 国务院
{代码} 股东大会 议案
```

### 常见陷阱

1. **利好出尽是利空**:重大利好公告前股价已经涨完,公告日反而高开低走
2. **重组并购的概率陷阱**:90% 的 A股重组都失败或夭折,炒重组要做最坏打算
3. **业绩超预期但被股价 price-in**:涨幅已大的票,业绩超预期反而成兑现信号
4. **政策利好的时滞**:从政策发布到业绩兑现可能要 1-3 年,股价短期是"炒预期"
5. **小心"伪利好"**:如"签订意向性合作协议"实际可能没有金额、没有约束力

## 四维交叉验证(关键!)

四维必须**交叉验证**,任一维度看上去很好不代表整体看好。下面是几种典型模式:

### 模式 A:四维共振(最强信号)
- 大盘强 + 资金涌入 + 技术突破 + 消息催化
- → 加权得分通常 ≥ +1.5,可给"强看多",但概率仍 clamp 65%

### 模式 B:三正一负(对冲)
- 例:大盘强 + 资金涌入 + 技术突破 + **消息面有利空**
- → 强制置信度降一级,明确写"对冲点:消息利空可能引发短线兑现"

### 模式 C:基本面好但技术弱(陷阱)
- 大盘弱 + 资金流出 + 技术跌破 + 消息面利好
- → 概率不要高估,可能是"利好兑现 + 大盘拖累",不参与

### 模式 D:全面看空但消息突变(博反弹)
- 大盘弱 + 资金流出 + 技术弱 + **突发重磅利好**
- → 可小仓位试错,但严格止损,不能因为"消息好"就重仓抄底

### 模式 E:无任何信号(中性)
- 加权得分在 -0.3 ~ +0.3 之间
- → 直接给"观望"建议,不交易就是最好的交易

## 打分时的强制要求

1. **每个维度的打分必须列 1-3 条具体证据**,每条证据后跟数据来源 [n]
2. **不能凭印象给分**,必须查到具体数据
3. **数据缺失时打 0 分**,不能猜测;同时在 markdown 中标"数据缺失"
4. **对冲信号必须显式标出**,不能默默把它平均掉
5. **打分要保守**:有疑虑时往中性 0 靠拢,不要随便给 ±2
