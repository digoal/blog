# 典型场景应对手册

> 这是 SKILL.md 步骤 5 的扩展。当个股属于以下场景时,按场景特定规则覆盖通用框架。
> 写预测前先判断属于哪个场景,如果有匹配,**按本手册的特殊规则微调打分和操作建议**。

## 场景识别决策树

```
[今日是涨停板?]
    ├── 是 → [N 连板?]
    │       ├── 首板 → 场景1
    │       ├── 2-3 板 → 场景2(连板梯队)
    │       └── 4 板以上 → 场景2 + 高度板退潮风险
    │
    ├── 否 → [近 N 日跌幅 > 20%?]
    │       ├── 是 → 场景3(超跌反弹)
    │       └── 否 → 进入下一判断
    │
    └── [属于业绩公告窗口期?]
            ├── 是(1/4/7/10 月) → 场景4
            └── 否 → 进入下一判断
                    │
                    └── [近 3 日有重大政策利好?]
                            ├── 是 → 场景5
                            └── 否 → 用通用框架
```

## 场景1:首板涨停股次日预测

### 关键判断信号

| 信号 | 强(+) | 中(0) | 弱(-) |
|---|---|---|---|
| 封单金额 / 流通盘 | > 5% | 1-5% | < 1% |
| 封板时间 | 10:30 前早盘封 | 11:00-14:30 | 14:30 后尾盘封 |
| 是否一字板 / T 字板 | 一字板(全天封死) | T 字板(开盘封) | 普通涨停 |
| 同板块涨停数量 | ≥ 5 家(板块效应) | 2-4 家 | 仅本只 |
| 是否上龙虎榜 | 知名游资买入 | 一般营业部 | 无 / 砸盘 |
| 涨停原因 | 强催化(政策、订单、业绩) | 概念跟风 | 无明显原因 |

### 次日操作策略

```
┌─ 高开幅度 ─→  ┌─ 0-3% ─→ 计划低吸(中枢价附近)
│              ├─ 3-7% ─→ 观望,等回踩(分时跌破均价线再考虑)
│              ├─ > 7% ─→ 放弃追高
│              ├─ 平开 ─→ 看分时,有回封信号可参与
│              └─ 低开 ─→ 看分时,封板可博弈;持续走弱则放弃
```

**次日开盘后 30 分钟特殊关注:**
- 站稳分时均价线 = 强势
- 跌破分时均价线 = 弱势
- 量比 > 2 + 红盘 = 资金继续看多
- 量比 < 0.7 + 绿盘 = 接力意愿弱

### 量化调整(在通用打分基础上)

- 一字板 / T 字板:总概率 +5%
- 早盘封板 + 知名游资:总概率 +3%
- 尾盘抢板:总概率 -3%
- 炸板后回封:总概率 -5%
- 板块龙头首板:总概率 +3%
- 板块跟风首板(板块已 5+ 涨停):总概率 -3%

### 风险提示

- 首板溢价最高的是次日开盘半小时,之后大概率回落
- 高度板(板块最高板)次日反而可能成接力对象,而跟风板容易被弃
- "一日游"风险:首板高开 7% 但中途破板,常常以收阴跌停告终
- **T+1 制度下首板必须假设"次日跌停可以承受",仓位严控在 5%-10%**

## 场景2:连板股梯队选择

### 优选原则

| 优先级 | 选择标准 |
|---|---|
| 1 | 选**高度龙头**(板块最高板),不选低度跟风 |
| 2 | 选**主线方向**(当下市场主线),不选孤立异动 |
| 3 | 选**低位连板**(1-3 板),不选**高位连板**(5 板以上) |
| 4 | 选**有龙虎榜资金支持**的,不选无人接力的 |
| 5 | 选**首阴(连板后第一根阴线)企稳的**,不选连阴的 |

### 连板梯队阶段判断

```
[1-2 板:启动期]  → 资金试探,胜率高但赔率小
[3-4 板:加速期]  → 资金共识强,赔率最佳
[5-6 板:亢奋期]  → 风险开始累积,必须减仓
[7 板以上:见顶]   → 不参与,等炸板
```

### 风险红线

- **4 板以上必须减仓 50%**
- **6 板以上原则上不参与**(高度板炸板大概率退潮全市场)
- **高度板炸板** = 全市场退潮信号,**第二天减仓所有连板股**
- **晋级率突然 < 30%** = 情绪退潮,不要逆势博弈

### 量化调整

- 主线板块龙头连板:加权得分 +0.3
- 跟风连板(非板块龙一):加权得分 -0.2
- 5 连板以上:加权得分强制 -0.5(高度风险)
- 大盘走弱 + 高度板炸板:加权得分 -1.0(灾难日)

### 数据来源关键词

```
今日 高度板 连板 几板
{代码} 几连板 是否 龙头
{所属板块} 连板梯队 今日
近 3 日 高度板 晋级率
```

## 场景3:超跌反弹股

### 触发条件(必须同时满足)

- 近 5-20 个交易日累计跌幅 > 20%
- 收盘价站上 5 日均线(止跌信号)
- 当日放量大阳(量比 > 2)
- 大盘止跌企稳(连续 2 日上涨或横盘)
- 板块整体出现反弹迹象

### 反弹阶段判断

```
[第 1 天:试探性反弹]  → 试仓 1/4
[第 2-3 天:确认上涨] → 缩量回踩不破阳线低点 = 加仓
[第 4-5 天:加速反弹] → 主力快速拉升 = 持有
[第 6-7 天:衰竭期]   → 量能萎缩 = 减仓
[第 7 天后:回落期]   → 必须清仓
```

### 关键禁忌

- **跌破年线(250 日均线)的股票不抄底**(趋势完全破坏)
- **连续阴跌 3 日以上、无止跌信号不抄底**(下跌中继)
- **无量阴跌不抄底**(缩量下跌说明无买盘承接,继续下跌概率大)
- **业绩雷区不抄底**(业绩还未披露 / 已预亏)

### 量化调整

- 跌破年线:加权得分强制 < -1.0,直接放弃
- 第一次放量大阳(止跌信号):加权得分 +0.3
- 二次回踩成功:加权得分 +0.5
- 反弹超 5 天:加权得分 -0.3(衰竭)

### 操作纪律

- 反弹股仓位上限:**总资金的 10%**(高风险,不可重仓)
- 止损位:5 日均线(跌破即清仓)
- 目标位:前期下跌的 1/3 - 1/2 处通常是阻力
- 时间止损:7 个交易日内未到目标位 = 平仓

## 场景4:业绩公告窗口期(1/4/7/10 月)

### 窗口期关键日期

| 月份 | 关键事件 |
|---|---|
| 1 月 | 上年年报预告(亏损或大幅变化必披露,1/31 截止) |
| 4 月 | 一季报(4/30 截止) |
| 7 月 | 半年报预告(7/15 部分公司、8/31 全部) |
| 10 月 | 三季报(10/31 截止) |

### 三种应对策略

#### 策略 A:已披露超预期股
- 业绩超预期(实际同比 > +50% 且超券商预期):**次日大概率高开,顺势参与**
- 但要注意"利好出尽":股价此前已大涨 > 50%,公告日反而是兑现日
- 量化调整:消息面 +2,但若涨幅已大,大盘强 + 资金 + 技术任一变弱 = 不追

#### 策略 B:已披露暴雷股
- 业绩暴雷(亏损或同比 < -50%):**立即清仓,不要幻想反弹**
- 通常会经历 2-3 个跌停板,跌停打开前不进场
- 量化调整:消息面 -2,加权得分通常 < -1,直接放弃

#### 策略 C:未披露但接近披露窗口的股
- **不重仓,不持仓过披露日**
- 如果一定要持有,最多 5% 仓位
- 提前查一致预期(券商研报),如果分歧大就回避

### 业绩雷区识别(提前回避)

- 高商誉股(商誉占净资产 > 30%):年报季商誉减值风险大
- 高质押股(大股东质押率 > 80%):业绩暴雷可能引发平仓盘
- 应收账款 / 营收比 > 50%:回款风险大
- 经营性现金流持续为负:利润和现金对不上
- 频繁更换审计、会计政策变更:粉饰嫌疑

### 量化调整

- 业绩超预期 + 涨幅 < 30%:消息面 +2
- 业绩超预期 + 涨幅 > 50%:消息面 +1(注意兑现)
- 业绩暴雷:加权得分强制 -1.5,直接放弃
- 业绩窗口期未披露股:最大仓位 5%

## 场景5:政策利好板块

### 判断维度

| 维度 | 强 | 中 | 弱 |
|---|---|---|---|
| 政策级别 | 国务院 / 国家发改委 | 部委级(工信部、能源局) | 行业级(协会) |
| 受益逻辑 | 直接受益 + 业绩兑现 | 间接受益 | 概念关联 |
| 板块龙头反应 | 当日涨停 + 量比 > 3 | 大涨 5%+ | 微涨 < 3% |
| 持续性预期 | 长周期(3-5 年规划) | 中周期(1-2 年) | 短周期(1 季度) |

### 操作节奏

```
[第 1 天:政策发布日]
   → 主线龙头涨停,跟风股大涨
   → 操作:追板块龙头(高开 < 5% 可买,集合竞价是关键)

[第 2 天:发酵日]
   → 看持续性
   → 缩量调整(主力没出货) = 加仓信号
   → 巨量阴跌(主力兑现) = 减仓信号

[第 3 天:分化日]
   → 强者恒强:龙头继续封板,跟风死
   → 操作:留龙头,砍跟风

[第 4-7 天:延续/衰退]
   → 量能持续放大 + 龙头不断高 = 继续持有
   → 量能萎缩 + 龙头滞涨 = 全部清仓

[1 周后:政策效应衰退]
   → 回归常态,等下一个政策催化
```

### 著名"政策市"案例提示

- **2024 年 9 月降准降息 + 政治局会议** → 大盘 1 周涨 30%
- **2024 年低空经济政策** → 万丰奥威等 5 倍股
- **2024 年人形机器人** → 持续 6 个月主线
- **2023 年 AI 行情** → 寒武纪、剑桥科技等数倍

### 量化调整

- 国家级政策利好:消息面 +2
- 部委级:消息面 +1
- 已经持续 1 周以上的政策:消息面 +1(炒新不炒旧)
- 已经持续 1 个月以上:消息面 0 或 -1(尾段风险)

## 通用风险提示(所有场景共用)

1. **任何场景下都遵守"单笔亏损 ≤ 2%"的底线**
2. **T+1 制度永远生效**,买入后必须假设次日跌停可承受
3. **场景判断不准 → 用通用框架兜底**,不要硬套场景
4. **多场景叠加时取最保守的**(如"高度板 + 政策利好" → 取高度板的退潮风险约束)
5. **大盘走弱时,所有场景的概率都要下调 5-10%**(系统性风险)
