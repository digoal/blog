## 一、加权得分计算

```
加权得分 = 大盘环境分 × 0.30 + 资金行为分 × 0.30 + 技术面分 × 0.25 + 消息面分 × 0.15

四维打分范围:每维 -2 ~ +2 整数
加权得分范围:-2.0 ~ +2.0(可有小数)
```

### 示例计算

| 维度 | 打分 | 权重 | 加权 |
|---|---|---|---|
| 大盘环境 | +1 | 0.30 | +0.30 |
| 资金行为 | +2 | 0.30 | +0.60 |
| 技术面 | 0 | 0.25 | 0.00 |
| 消息面 | +1 | 0.15 | +0.15 |
| **合计** | | | **+1.05** |

→ +1.05 落在 ">+1.0" 区间 → 强看多(置信度 高)

## 二、加权得分 → 方向 + 置信度

| 加权得分区间 | 方向 | 置信度 | 操作建议 |
|---|---|---|---|
| > +1.5 | 极强看多 | 高 | 重点参与,可加仓 |
| +1.0 ~ +1.5 | 强看多 | 高 | 标准参与 |
| +0.3 ~ +1.0 | 弱看多 | 中 | 试探性买入 |
| -0.3 ~ +0.3 | 中性震荡 | 中低 | 观望或日内交易 |
| -1.0 ~ -0.3 | 弱看空 | 中 | 减仓或回避 |
| -1.5 ~ -1.0 | 强看空 | 高 | 清仓 |
| < -1.5 | 极强看空 | 高 | 立即清仓,可考虑融券 |

**对冲信号处理**:置信度强制降一级。如出现下列情况:
- 大盘强 + 个股资金流出
- 消息利好 + 技术跌破
- 资金涌入 + 业绩预亏

## 三、加权得分 → 方向概率

### 基础公式

```
基础概率 = 50%(A股次日方向接近随机)
偏移 = 加权得分 × 7.5%
初步概率 = 50% + 偏移
```

| 加权得分 | 偏移 | 初步概率 |
|---|---|---|
| +2.0 | +15% | 65% |
| +1.5 | +11.25% | 61% |
| +1.0 | +7.5% | 57.5% |
| +0.5 | +3.75% | 53.75% |
| 0 | 0% | 50% |
| -0.5 | -3.75% | 46.25% |
| -1.0 | -7.5% | 42.5% |
| -1.5 | -11.25% | 39% |
| -2.0 | -15% | 35% |

### 微调规则

1. **同板块龙头明确强势**:+3%
2. **同板块龙头明确弱势**:-3%
3. **有清晰 P0 催化 + 四维同向**:±5%
4. **集合竞价异动符合预期**(高开 1-3%):+2%
5. **信息严重缺失**:概率 clamp 回 45%-55% 之间

### 涨停股特殊处理(场景1)

如果今日是涨停板,次日方向概率额外调整:
- 一字板 / T 字板(全天封死):次日方向概率上调 +5%(但需考虑高开幅度风险)
- 早盘涨停(10:30 前)且封单 > 流通盘 5%:+3%
- 尾盘抢板(14:45 后):-3%(诱多概率高)
- 炸板后回封:-5%(分歧已现)

### 硬性 clamp(无条件执行)

```
最终概率 = clamp(微调后概率, 35%, 65%)
```

**任何 > 65% 的概率自动 clamp 到 65%。** A股次日方向是低信噪比事件,过度自信 = 爆仓最快路径。

### 概率区间(80% 置信)

最终概率给出区间,体现不确定性:
```
区间下界 = 最终概率 - 5%
区间上界 = 最终概率 + 5%
```

例:最终概率 58% → 区间 53% - 63%

## 四、加权得分 → 价格区间

### ATR(平均真实波幅)简化估算

```
ATR_5 = (近5日 最高 - 近5日 最低) 的平均值

更精确版(可选):
真实波幅 = max(当日最高-最低, 当日最高-昨收, 昨收-当日最低)
ATR_5 = 近5日 真实波幅 的平均值
```

### 价格区间计算

```
中枢价 = 今日收盘价 × (1 + 加权得分 × 1.5%)

上界 = 中枢价 + ATR_5 × 1.28
下界 = 中枢价 - ATR_5 × 1.28
(1.28 = 80% 置信的正态分布分位数)
```

### 涨跌停板硬约束(必须应用)

不同板块的涨跌停幅度:
- 主板(60xxxx、00xxxx):±10%
- ST 股:±5%
- 创业板(30xxxx)、科创板(68xxxx):±20%
- 北交所(43xxxx、83xxxx、87xxxx、92xxxx):±30%

```
涨停价 = 今日收盘价 × (1 + 涨幅限制)
跌停价 = 今日收盘价 × (1 - 涨幅限制)

最终上界 = min(理论上界, 涨停价)
最终下界 = max(理论下界, 跌停价)
```

### 示例

贵州茅台(600519,沪市主板,±10%)
- 今日收盘:1875.50 元
- 加权得分:+1.05
- 近 5 日 ATR:42 元

```
中枢价 = 1875.50 × (1 + 1.05 × 1.5%) = 1875.50 × 1.01575 = 1905.04 元
上界 = 1905.04 + 42 × 1.28 = 1905.04 + 53.76 = 1958.80 元
下界 = 1905.04 - 42 × 1.28 = 1905.04 - 53.76 = 1851.28 元

涨停价 = 1875.50 × 1.10 = 2063.05 元(上界未超)
跌停价 = 1875.50 × 0.90 = 1687.95 元(下界未触)

最终价格区间:1851 - 1959 元
中枢:1905 元(较收盘 +1.6%)
```

## 五、仓位计算(强制公式)

### 单笔风险预算

```
单笔最大可承受亏损 = 账户净值 × 2%
```

### 仓位公式

```
入场价 - 止损价 = 单股最大亏损
仓位金额 = (账户净值 × 2%) ÷ (入场价 - 止损价) × 入场价
仓位股数 = (仓位金额 ÷ 入场价) 向下取整到 100 的倍数(A股 1 手 = 100 股)
仓位占比 = 仓位金额 ÷ 账户净值 × 100%
```

### 示例

账户 100 万,贵州茅台预期入场 1875 元,止损 1830 元
```
单股最大亏损 = 1875 - 1830 = 45 元
仓位股数(理论) = (1,000,000 × 2%) ÷ 45 = 444.44 股
仓位股数(取整) = 400 股(向下取整到 100 的倍数)
仓位金额 = 400 × 1875 = 750,000 元
仓位占比 = 75%
```

### 仓位上限(置信度约束)

| 置信度 | 单笔仓位上限 |
|---|---|
| 高 | ≤ 总资金 30% |
| 中 | ≤ 总资金 15% |
| 低 | ≤ 总资金 5% |

上述示例 75% 超过"高"置信度的 30% 上限,所以**仓位强制压缩到 30%**:
```
最终仓位金额 = 1,000,000 × 30% = 300,000 元
最终股数 = 300,000 ÷ 1875 = 160.0 → 取 100 股(向下到整百)
实际仓位金额 = 100 × 1875 = 187,500 元
实际仓位占比 = 18.75%
```

### 现金底线

**永远保留至少 20% 现金**,应对:
- 极端行情突发暴跌
- 突发更好的机会
- T+1 制度下次日补仓的灵活性

## 六、目标价计算(赔率 ≥ 2:1 才参与)

### 目标价 TP1(赔率 1:1,平 50% 仓位)
```
TP1 = 入场价 + (入场价 - 止损价)
```

### 目标价 TP2(赔率 2:1,再平 30% 仓位)
```
TP2 = 入场价 + 2 × (入场价 - 止损价)
```

### 跟踪止损(剩余 20% 仓位)
```
跟踪止损 = 当日最高价 - 1 × ATR_5
(每日动态上移,锁定利润)
```

### 示例

贵州茅台,入场 1875,止损 1830
- 风险 = 1875 - 1830 = 45 元
- TP1 = 1875 + 45 = 1920 元(平 50%)
- TP2 = 1875 + 90 = 1965 元(再平 30%)
- 跟踪:剩余 20%,每日按"最高 - 42 元"动态止盈

## 七、时间止损(A股专属)

```
最大持仓天数:
- 高置信度:3 个交易日
- 中置信度:2 个交易日
- 低置信度:1 个交易日(日内)

到期未触发 TP1 = 平仓
```

**特别规则**:
- 周五尾盘必须减仓 50%(不持仓过周末,周末黑天鹅风险)
- 重大事件前(财报、政策窗口)必须减仓
- 连续 2 日横盘未上涨 = 平仓(资金可能在悄悄出货)

## 八、止损条件(任一触发立即执行)

| 触发条件 | 操作 |
|---|---|
| 跌破设定止损价 | 全部清仓 |
| 跌破当日最低 | 短线止损(日内) |
| 跌破 5 日均线且收阴 | 减仓 50% |
| 跌破 10 日均线 | 全部清仓 |
| 单日亏损达 7% | **无条件清仓**,不解释 |
| 出现重大利空公告 | 立即清仓 |
| 大盘跌破年线 | 全部清仓 |

## 九、完整决策流程(把上面公式串起来)

```
输入:用户给的股票代码

Step 1: 标的解析 → 确认板块、涨跌停幅度
Step 2: 搜证 → 收集四维数据
Step 3: 四维打分 → 得到加权得分(-2 ~ +2)
Step 4a: 加权得分 → 方向 + 置信度
Step 4b: 加权得分 → 概率(clamp 35-65%)
Step 4c: 加权得分 → 价格区间(受涨跌停板硬约束)
Step 5a: 入场价 = 价格区间中枢 或 重要支撑位上方
Step 5b: 止损价 = 关键支撑下 1-2%
Step 5c: TP1 = 入场 + 风险;TP2 = 入场 + 2×风险
Step 5d: 仓位金额 = 账户×2% ÷ (入场-止损) × 入场,再 clamp 到置信度上限
Step 5e: 时间止损 = 置信度对应天数
Step 6: 写出 markdown 报告
Step 7: 输出摘要
```
