# A股搜证查询模板库

> 这是 SKILL.md 步骤 2 的展开。所有搜证查询都按"情境 → 模板"组织,直接复制改 {代码} 即可。
> 写预测前先读本文件,挑选适合当前股票情境的查询组合。
> **强制规则:必须使用 `mcp__MiniMax__web_search`,禁止使用内置 `web_search`。**

## 查询基本原则

1. **每次查询 1-2 个关键词,不要堆砌**
   - 错: `{代码} 今日 龙虎榜 北向 主力 融资 公告` (噪声太大)
   - 对: `{代码} 今日 龙虎榜`
2. **每次查询读 3-5 条结果,做交叉验证**(单一来源 = 瞎猜)
3. **时效性优先,取最近 24 小时数据**(把日期带进关键词)
4. **数据冲突时,权威源优先**:东方财富、同花顺、巨潮资讯网、上交所 / 深交所公告 > 自媒体

## 一、大盘环境查询(P0)

```
今日 上证指数 收盘 涨跌幅
今日 创业板指 收盘 涨跌
今日 科创50 收盘
今日 涨停 跌停 家数 A股
今日 主线 板块 涨停板
今日 高度板 连板梯队
今日 沪深两市 成交额 万亿
北向资金 今日 净流入 净流出
近一周 涨停股 次日 红盘率
申万 行业 今日 涨跌幅 排行
```

**判断模板:**
- 上证指数 + 5 日均线状态(在上 = 强,跌破 = 弱)
- 涨停家数 + 跌停家数(涨 80+ 强,跌 50+ 弱)
- 主线连板高度(3 板以上有龙头,断板就退潮)
- 北向资金(净流入 50 亿+ 强,净流出 50 亿+ 弱)

## 二、个股核心数据查询(P0)

```
{代码} 今日 收盘价 涨跌幅
{代码} 今日 成交量 换手率
{代码} 今日 龙虎榜
{代码} 主力资金 流入 今日
{代码} 近一周 主力净流入
{代码} 近 5 日 K线 走势
{代码} 当前 流通市值 总市值
{代码} {股票名} 公告 最新
```

**判断模板:**
- 当日涨跌幅(已涨 > 7% 不追;跌 > 5% 看反弹机会)
- 龙虎榜(有上榜 = 信号强;游资买入 vs 砸盘)
- 主力净流入 / 流通市值(> 1% 强势;< -1% 弱势)
- 近 5 日是阴线还是阳线占多?

## 三、龙虎榜深度查询(P0/P1)

```
{代码} 今日 龙虎榜 买入 席位
{代码} 今日 龙虎榜 卖出 席位
{代码} 龙虎榜 知名游资
{代码} 龙虎榜 机构席位
{代码} 龙虎榜 净买入 金额
{游资席位名} 今日 买入 哪些股
```

**判断模板:**
- 净买入金额(> 5000 万强;<1000 万弱)
- 买入席位(知名游资 强;一日游游资 弱;机构席位 中长线)
- 卖出席位(若是昨日买入的游资卖出 = 接力砸盘,警惕)
- "机构专用"出现在卖一 = 危险信号

**知名游资席位(部分代表)**:
- 中国银河绍兴(章盟主)
- 国泰君安上海打浦路(章盟主)
- 华鑫证券上海分公司(陈小群)
- 东方财富拉萨(游资聚集)
- 深股通专用 / 沪股通专用(北向)

## 四、资金面查询(P1)

```
{代码} 北向资金 持股比例 变化
{代码} 北向 加仓 减仓 近期
{代码} 融资融券 余额 最新
{代码} 融资净买入 近 5 日
{代码} 大单 净买入 今日
{代码} 主力 出货 流入 信号
{代码} 限售股 解禁 时间表
{代码} 大股东 持股 变化
```

**判断模板:**
- 北向持股比例环比变化(连续 3 日加 = 强信号)
- 融资余额(放大 = 杠杆看多;骤降 = 恐慌减仓)
- 限售解禁(次日有解禁 = 大利空,直接放弃)

## 五、技术面查询(P1)

```
{代码} 5日均线 10日均线 20日均线 当前
{代码} 量比 换手率 今日
{代码} 近 5 日 最高价 最低价
{代码} 前期 高点 压力位
{代码} 前期 低点 支撑位
{代码} K线形态 技术分析
{代码} MACD KDJ 当前
```

**判断模板:**
- 均线排列(多头排列 强;空头排列 弱)
- 量比(> 2 放量;< 0.7 缩量)
- 当前价 vs 关键位(突破前高 强;跌破前低 弱)
- ATR 估算:(近5日最高 - 近5日最低) 的平均值

```
{代码} 商誉减值 风险
{代码} 证监会 立案 调查
```

**判断模板:**
- 业绩公告(超预期 +;低于预期 -)
- 减持公告(任何减持都是 -1 到 -2)
- 解禁公告(必须避开)
- 重组进展(谨慎,90% 失败)
- 立案 / 警示函(直接清仓)

## 七、板块联动查询(P1)

```
{所属行业} 今日 涨幅 排行
{所属概念} 今日 龙头 涨幅
{所属行业} 近 3 日 资金流向
{所属概念} 主线 持续性
近 3 日 {行业} 政策 消息
{行业} 板块 ETF 走势
{行业} 龙头股 今日 表现
```

**判断模板:**
- 同板块龙头是涨停 / 大涨 / 平盘 / 大跌?
- 板块整体强势 = 个股加分;板块走弱 = 个股减分
- 政策消息(国家级 > 部委级 > 行业级)

## 八、宏观环境查询(P2)

```
今日 A股 重要事件 财经日历
明日 A股 政策预期 央行
富时A50 期指 隔夜 走势
人民币汇率 中间价 今日
今日 港股 收盘 涨跌(联动信号)
美股 隔夜 收盘(对中概股、AI 等板块联动)
LPR 利率 最新调整
PMI CPI PPI 数据 最近
央行 公开市场 操作 今日
```

**判断模板:**
- 富时 A50 期指隔夜涨 = 次日开盘大概率高开
- 人民币贬值 = 出口板块利好,进口板块利空
- 央行降息 / 降准 = 大盘利好(尤其是金融、地产)

## 十、搜证质量自检

每次搜证完成后,自检:

- [ ] 至少 6 条 P0 查询都做了?
- [ ] 每条查询至少看 3 条结果?
- [ ] 大盘数据有没有?(指数、涨跌停家数、北向)
- [ ] 龙虎榜有没有查?(决定资金面打分的关键)
- [ ] 公告有没有查?(避开雷区)
- [ ] 板块联动有没有查?(决定加减分)
- [ ] 数据冲突时,选择了权威源?
- [ ] 信息严重缺失时,降级了置信度?

**搜证不够 = 后面的打分全是凭空捏造,等同于占卜。**

## 十一、搜证省时技巧

1. **批量查询**:大盘数据一次查"今日 A股 大盘 涨跌停 涨跌家数 北向",一次拉所有大盘信号
2. **优先东方财富**:东财个股页基本覆盖了 80% 的需求(K线、龙虎榜、资金、公告、解禁)
3. **巨潮资讯网**:公司公告原文,优于二手解读
4. **同花顺 i 问财**:可以问"贵州茅台 近 5 日 主力资金"获得汇总
5. **不要重复查**:同一数据已经在某条结果中确认,不必再查;省下查询额度查关键的
