## 一、交易制度类

| 术语 | 小白解释 |
|---|---|
| **T+1** | 今天买的股票,明天才能卖。是A股的基本交易制度。这意味着一旦买错,你被锁一晚上,中间发生任何事(暴跌、停牌)都没法跑。 |
| **涨停板** | 单日股价最多能涨多少。主板 ±10%,创业板 / 科创板 ±20%,ST 股 ±5%,北交所 ±30%。涨到顶就不能再涨了。 |
| **跌停板** | 同上,单日股价最多能跌多少。 |
| **ST 股** | 股票名字前面有 ST 或 *ST,表示公司被风险警示(财务异常、可能退市)。涨跌幅限制 ±5%,风险大。 |
| **退市** | 公司不再在交易所交易,股票变成废纸或转到三板市场。 |
| **停牌** | 股票暂停交易。可能是公司主动(重大事项)或交易所要求(异常波动核查)。复牌后可能大涨或大跌。 |
| **集合竞价** | 9:15-9:25 这 10 分钟,所有人挂单不成交,9:25 统一撮合一次,形成开盘价。可以看到次日开盘大致方向。 |
| **连续竞价** | 9:30-11:30 和 13:00-15:00,正常交易时段。 |
| **尾盘** | 14:30-15:00。这段时间机构调仓、资金抢筹 / 出货最明显,是次日方向的重要线索。 |
| **一手** | A股最小交易单位 = 100 股。买股票必须按 100 的倍数买。 |

## 二、K线技术类

| 术语 | 小白解释 |
|---|---|
| **K线** | 一根代表当日(或当周、当月)的"价格小柱子"。红色 = 上涨,绿色 = 下跌。包含开盘、收盘、最高、最低 4 个价格。 |
| **均线 / MA** | 把过去 N 天的收盘价取平均,画成一条线。5 日均线最敏感,250 日均线(年线)最稳。 |
| **多头排列** | 短期均线(5日)在中期(10日)上面,中期在长期(20日)上面,长期在更长(60日)上面,层层向上 → 强势趋势。 |
| **空头排列** | 与多头排列相反,均线层层向下 → 弱势趋势。 |
| **量比** | 当日成交量 / 近 5 日平均成交量。> 2 = 放量(有大资金动作);< 0.7 = 缩量。 |
| **换手率** | 当日成交量 / 流通股本。换手率高 = 短期资金活跃;低 = 不活跃。 |
| **ATR(平均真实波幅)** | 简化理解:近 5 日股价的平均波动范围,用来衡量这只股每天能涨跌多少元。 |
| **支撑位** | 股价下跌时容易反弹的位置(通常是前期低点、密集成交区、重要均线)。 |
| **压力位 / 阻力位** | 股价上涨时容易回落的位置(通常是前期高点、整数关口)。 |
| **突破** | 股价向上穿过压力位 → 多头确认。带量突破才有效,无量突破易"假突破"。 |
| **跌破** | 股价向下穿过支撑位 → 空头确认。 |
| **金叉** | 短期均线上穿长期均线 → 看多信号。如 5 日上穿 10 日。 |
| **死叉** | 短期均线下穿长期均线 → 看空信号。 |
| **缺口** | K线上的"跳空",今日开盘价比昨日收盘价高(向上缺口)或低(向下缺口)。缺口一般会被回补。 |

## 三、资金行为类

| 术语 | 小白解释 |
|---|---|
| **主力资金** | 大单交易资金(通常指 100 万元以上的单笔交易)。被认为是机构、游资等大资金。 |
| **散户资金** | 小单交易资金(通常 < 5 万元)。 |
| **大单 / 特大单** | 大单 = 单笔 100 万 - 500 万;特大单 = 单笔 > 500 万。 |
| **主力净流入** | 主力买入金额 - 主力卖出金额。净流入大 = 大资金在买;净流出大 = 大资金在卖。 |
| **北向资金 / 陆股通** | 海外资金通过香港买入 A股的资金。被称为"聪明钱",虽然不完全准确,但是重要参考。 |
| **南向资金** | 内地资金通过港股通买入港股,与北向相反。 |
| **融资余额** | 投资者向券商借钱买股票,目前还欠的总额。融资余额放大 = 杠杆资金看多;骤降 = 恐慌减仓。 |
| **融券余额** | 投资者向券商借股票卖出(做空),目前还欠的股票数 × 市价。 |
| **两融** | 融资融券的简称。能融资融券的股票叫"两融标的"。 |
| **封单** | 涨停板上挂的买入委托(买入封单)或跌停板上挂的卖出委托(卖出封单)。买入封单大 = 涨停越牢固。 |
| **撤单** | 把之前挂的委托撤回。涨停板上撤单多 = 主力意图减弱,可能炸板。 |
| **炸板** | 涨停板被巨量卖单砸开。是分歧加大的信号,次日大概率走弱。 |

## 四、龙虎榜与游资类

| 术语 | 小白解释 |
|---|---|
| **龙虎榜** | 交易所每日盘后公布的异常波动股名单,显示当日买入卖出最多的 5 家券商营业部 / 机构席位。是最重要的资金面信号。 |
| **席位** | 龙虎榜上的"营业部 + 营业地点"名称。每个席位背后可能是一个游资。 |
| **游资** | 民间大资金,以打板、追热点为生的短线高手。著名游资有章盟主、陈小群、方新侠、量化打板等。 |
| **机构席位** | 龙虎榜显示"机构专用"字样的席位。代表公募、保险、社保等长线资金。 |
| **打板** | 主动买涨停板的策略,赚次日溢价。 |
| **接力** | 前一日涨停的票,次日有人继续买入推高(可能再涨停)。 |
| **首板** | 第一个涨停板。 |
| **N 连板** | 连续 N 个交易日涨停。如"3 连板"= 连续 3 天涨停。 |
| **高度板** | 当日市场上连板数最高的股票(如全市场最高 5 连板)。是市场情绪温度计。 |
| **晋级率** | 前一日 N 连板的票,有多少在次日继续涨停。晋级率高 = 情绪高;晋级率低 = 情绪退潮。 |
| **打板率** | 涨停股次日开盘溢价 / 触及涨停的概率。是衡量短线赚钱效应的指标。 |
| **一日游游资** | 风格快进快出的游资,买入次日就卖。跟这种游资风险大。 |
| **接力游资砸盘** | 前一日买入推高的游资,第二日反手卖出。是退潮信号。 |

## 五、板块与行情类

| 术语 | 小白解释 |
|---|---|
| **板块** | 把股票按行业(食品饮料、电池、医药)或主题(AI、机器人、低空经济)分组。 |
| **概念 / 题材** | 比"行业"更细的标签,通常基于热点话题(如"低空经济"、"AI 算力")。 |
| **主线** | 当下市场最强、连板最多的板块。市场永远只奖励主线龙头。 |
| **龙头股** | 板块里走势最强、连板数最高的股票。市场关注度最高。 |
| **跟风股** | 跟着龙头一起涨,但强度差很多。次日大概率走弱。 |
| **轮动** | 资金从一个板块流到另一个板块。"快速轮动"= 市场没有持续性主线。 |
| **风格切换** | 大盘股 / 小盘股、价值股 / 成长股之间的资金切换。 |
| **赚钱效应** | 大多数股票都在涨,涨停板能晋级 → 散户敢进场。 |
| **亏钱效应** | 大多数股票都在跌,涨停板次日就破板 → 散户不敢进场。 |
| **千股跌停** | 市场恐慌时,跌停股超过 1000 家。极端弱势信号。 |
| **千股涨停** | 市场极度亢奋时,涨停股超过 1000 家。极端强势信号(也常是顶部)。 |

## 六、公告与基本面类

| 术语 | 小白解释 |
|---|---|
| **业绩预告** | 上市公司在正式财报前提前预告业绩(亏损或盈利大幅变化时必须披露)。窗口期 1/4/7/10 月。 |
| **业绩快报** | 正式财报前的"预览版",数据未审计但接近最终值。 |
| **季报 / 年报** | 季度报告 / 年度报告。包含详细财务数据。一季报、半年报、三季报、年报。 |
| **EPS(每股收益)** | 公司净利润 / 总股本。衡量赚钱能力的核心指标。 |
| **业绩超预期** | 实际 EPS 比券商一致预期高 > 10%。利好。 |
| **业绩暴雷** | 业绩远低于预期,或大幅亏损。利空,次日开盘大概率跌停。 |
| **商誉减值** | 公司并购时溢价部分需要计入"商誉",当被收购方业绩不达预期时,要把商誉账面价值打折扣 → 直接吞噬利润。年报季高发。 |
| **限售股解禁** | 原本不能交易的股票(如大股东持有)到了约定时间可以卖出。解禁后流通盘扩大,有抛压。 |
| **大股东减持** | 持股 5% 以上的股东卖出股票。是利空信号。 |
| **股份回购** | 公司用自有资金从二级市场买回自家股票(注销或留作激励)。表示公司对自己有信心,利好。 |
| **定增 / 增发** | 上市公司发行新股募资。短期利空(摊薄股权)。 |
| **重组 / 并购** | 公司收购或被收购、合并、重大资产置换。利好,但 90% 失败,要谨慎。 |
| **立案调查** | 证监会对公司违规行为立案。重大利空,大概率跌停,可能停牌。 |
| **巨潮资讯网** | 中国证监会指定的上市公司公告披露平台,看公告原文必去。 |

## 七、交易策略类

| 术语 | 小白解释 |
|---|---|
| **入场价** | 计划买入的价位。 |
| **止损价** | 跌到这个价位必须卖出,锁定最大亏损。 |
| **止盈价 / 目标价** | 涨到这个价位卖出,锁定利润。 |
| **TP1 / TP2** | Take Profit 1 / 2。第一目标位 / 第二目标位,分批止盈。 |
| **赔率 / 风险报酬比** | (目标价 - 入场价)/(入场价 - 止损价)。≥ 2:1 才值得参与。 |
| **仓位** | 用了多少资金。10% 仓位 = 用了 10% 的总资金。 |
| **重仓** | > 30% 仓位。 |
| **轻仓 / 试仓** | < 10% 仓位,小钱试错。 |
| **加仓** | 在原仓位基础上再买。 |
| **减仓** | 卖出一部分。 |
| **清仓** | 全部卖出。 |
| **空仓** | 持有 100% 现金,不买任何股票。 |
| **抄底** | 在大跌后买入,赌反弹。常常"接飞刀"被套。 |
| **追高** | 在已经大涨后买入,赌继续涨。常常买在顶部。 |
| **斩仓** | 主动止损,认输离场。 |
| **被套** | 买入后跌破成本价。 |
| **解套** | 反弹回成本价,可以保本卖出。 |
| **死扛** | 被套后不止损,赌反弹。最容易爆仓的行为。 |
| **跟踪止损** | 随着股价上涨动态上移止损位,锁定利润。 |
| **时间止损** | 持仓到一定天数仍无收益就强制平仓。 |
| **金字塔加仓** | 试仓 1/4 → 确认后加 1/4 → 再加 1/4 → 永远保留 1/4 不动用。 |
| **马丁加仓(反金字塔)** | 越跌越加。极度危险,不推荐。 |
| **量价配合** | 价格上涨时成交量也放大(健康);价格上涨但成交量萎缩(警惕)。 |
| **量价背离** | 价格创新高但成交量没创新高,或反之。通常预示反转。 |
| **价托量** | 价格平稳但成交量持续放大,可能是主力建仓。 |

## 使用规则

1. **写预测时,术语第一次出现立即用括号给小白解释**。
2. **如果一个段落里有 3 个以上术语,改用通俗语言重写**,而不是把所有术语都注解一遍。
3. **遇到本词典没有的术语,自己用 1-2 句话解释**,默认读者完全不懂。
4. **最后"专业术语小白解释"章节里再统一列出所有用到的术语**,方便查阅。
