# 分析框架与推演方法论

本文件是 6 维分析的详细方法论, 为 SKILL.md 的工作流提供理论支撑。

---

## 维度 1 — 产业链与利润分布

### 1.1 产业链结构画法

**第一步: 锚定边界**
- 最上游(原材料/底层技术/IP) → 中游(制造/加工/集成) → 下游(渠道/品牌/服务) → 终端(消费者/使用场景)
- 边界不明确时, 问自己: "用户付钱买的是什么?" 顺着钱的流向反推

**第二步: 拆出节点**
- 每个节点问三件事: 1) 提供什么 2) 谁是代表企业 3) 毛利率大约多少
- 节点切分粒度: 不能太粗(否则看不到利润洼地), 不能太细(否则淹没主线)
- 经验值: 5-9 个节点最佳

**第三步: 识别卡脖子位置**
- 哪个节点一旦断供, 整条链就瘫痪?
- 哪个节点有定价权, 可以向上下游转嫁成本?
- 卡脖子点 = 最有话语权的位置 = 最高利润点 (但也可能是政策/地缘风险点)

### 1.2 利润分布的两种画法

**画法 A — 微笑曲线** (施振荣)
- 横轴: 产业链节点 (左到右 = 上游到下游)
- 纵轴: 毛利率/单位利润
- 形态: U 形或 J 形, 两端高(研发+品牌)、中间低(制造组装)
- 例: 苹果手机产业链 — 芯片设计(高通/台积电, 50%+) vs 组装(富士康, 7%)
- 例: 咖啡产业链 — 种植(1-2%) vs 烘焙(20-30%) vs 品牌零售(30-50%)

**画法 B — 价值链** (迈克尔·波特)
- 横轴: 主要活动(进料物流→生产→出货物流→营销销售→服务)
- 横轴: 支持活动(基础设施、人力资源、技术开发、采购)
- 找价值活动中的"溢价点"和"成本驱动因素"

### 1.3 节点分析三件套
每个产业链节点, 必须给出:
- **代表企业 ≥ 1 家**: 名字+一句话定位+关键数据(市占率/营收/毛利)
- **集中度**: CR3/CR5/CR10, 知道这是分散市场还是寡头市场
- **关键变量**: 这个节点的玩家竞争什么? (成本? 技术? 渠道? 品牌?)

## 维度 2 — 供需与连接

### 2.1 供给侧 (Supply)

| 维度 | 要回答的问题 |
|---|---|
| **产能** | 现有产能多少? 开工率? 未来 3 年新增产能? |
| **玩家** | 头部是谁? 集中度? 新进入者? 退出者? |
| **技术** | 主流技术路线? 替代技术? 技术成熟度? |
| **成本结构** | 主要成本项? 受什么波动影响? (原材料、能源、人力) |
| **政策** | 准入门槛? 牌照? 补贴? 环保? |

### 2.2 需求侧 (Demand)

| 维度 | 要回答的问题 |
|---|---|
| **规模** | 总市场规模? 增速? 渗透率? |
| **人群** | 用户画像 (年龄/地域/收入/使用场景) |
| **频次** | 高频/低频? 客单价? LTV? |
| **驱动** | 增长驱动是什么? (人口/政策/技术升级/消费升级) |
| **天花板** | 3-5 年后这个市场的天花板? |
| **替代** | 用户的钱还会花在哪里? 替代品/竞品 |

### 2.3 连接侧 (Connection)

"连接"是供需之间的桥梁, 也是最容易出机会的地方:

- **渠道**: 线下门店? 电商? 经销? 直营? 平台?
- **信息流**: 用户怎么知道这个产品? (搜索/推荐/口碑/广告)
- **物流**: 履约方式? 即时配送/快递/自提/到店?
- **资金流**: 账期? 预付? 分期? 信用?
- **平台**: 是否有平台型玩家在中间抽佣? (如美团/淘宝/抖音)

**关键问题**: "连接侧是否被某个巨头垄断? 垄断带来的不合理是什么?"

## 维度 3 — 人货场 (零售三要素)

"人货场"源自零售业, 但适用于任何 C 端行业, 也可借用于 B 端场景。

### 3.1 人 (People)

- **人口学**: 年龄/性别/收入/职业/地域
- **心理画像**: 价值观/生活方式/消费动机
- **决策路径**: AIDA / 5A 模型 / Customer Journey
- **用户分层**:
  - 头部用户 (高客单/高复购) — 利润来源
  - 腰部用户 (主流) — 规模来源
  - 尾部用户 (低价值) — 流量来源
- **未满足需求**: 用户现在怎么解决问题? 痛点是什么?

### 3.2 货 (Goods)

- **产品矩阵**: 爆款 / 利润款 / 形象款 / 引流款
- **SKU 逻辑**: 宽度 (覆盖人群) vs 深度 (单品销售) 的取舍
- **价值主张**: 这个产品为什么存在? 替代了什么?
- **定价层级**: 锚定效应 — 用户怎么感知价值?
- **迭代节奏**: 一年几次迭代? 跟随还是引领?

### 3.3 场 (Place/Scene)

- **触点**: 用户在哪里遇到这个产品? (小红书/抖音/电梯/朋友推荐)
- **场景**: 使用的物理和情绪场景 (早餐咖啡 vs 下午奶茶 vs 夜宵小龙虾)
- **转化路径**: 触点 → 兴趣 → 决策 → 购买 → 复购
- **体验地图**: 从认知到分享的全链路关键节点
- **场域创新**: 新的消费场景在哪里? (露营咖啡/露营烧烤/车载小冰箱)

## 维度 4 — 代表企业分析

每个代表企业分析 4 个层面:

### 4.1 商业模式
- 收入来源 (卖什么/卖给谁/怎么收钱)
- 成本结构 (主要成本项)
- 关键资源 (数据/渠道/牌照/技术/品牌)
- 护城河类型 (规模/网络效应/技术/品牌/成本/牌照)

### 4.2 核心竞争力
用 **VRIN 框架** 判断 (巴尼):
- V(Valuable): 有价值?
- R(Rare): 稀缺?
- I(Inimitable): 难以模仿?
- N(Non-substitutable): 不可替代?

### 4.3 财务表现
- 营收 / 净利润 / 增速
- 毛利率 / 净利率
- 经营性现金流
- ROE / ROIC
- 用户数 / 复购率 (互联网企业)

### 4.4 关键风险
- 政策风险
- 竞争加剧
- 技术替代
- 创始人风险
- 财务风险 (高负债/现金流)

## 维度 5 — 代表产品分析

### 5.1 价值主张 (Value Proposition)
- 这个产品解决谁的什么问题?
- 与替代方案相比, 优势是什么?
- 一句话定位: "X 行业的 Y, 给 Z 用的 W"

### 5.2 定价策略
- 成本定价 vs 价值定价 vs 竞争定价
- 锚定价格: 用户拿谁做对比?
- 价格分层: 引流款/主力款/利润款
- 价格变化: 提价能力? 降价空间?

### 5.3 用户口碑
- NPS / 净推荐值
- 复购率 / 流失率
- 评价关键词 (好评/差评)
- 社媒声量

### 5.4 迭代路径
- 版本演进: v1 → v2 → v3
- 关键功能增减
- 失败的尝试
- 下一版本方向

## 维度 6 — 推演方法论 (核心)

推演是本 skill 的灵魂, 必须严格按以下顺序。

### 6.1 找不合理点

**5 个找法**:

1. **信息不对称**: 谁掌握信息但没传递? 谁缺信息? (例: 中医产业链中, 患者不懂药材质量)
2. **效率低下**: 哪个环节周转慢/损耗大/重复劳动? (例: 鲜花从花农到终端加价 5 倍)
3. **利润失衡**: 哪个节点拿走了超过其价值贡献的利润? (微笑曲线两端 vs 中间)
4. **摩擦点**: 用户决策的阻力是什么? (例: 装修报价不透明, 用户难决策)
5. **政策套利**: 哪里有政策导致的扭曲? (例: 教培双减前的学科培训)

**每个不合理点, 写明**:
- 表现 (具体场景/数据)
- 根因 (为什么存在)
- 受害方 (谁最不爽)
- 持续条件 (什么情况下会持续存在)

### 6.2 推导机会点

每个不合理点 = 至少一个机会点。机会分类:

| 机会类型 | 描述 | 风险 |
|---|---|---|
| **效率提升型** | 用新工具/流程降低摩擦 | 教育用户成本 |
| **信息中介型** | 提供透明的对比/评价 | 难以建立信任 |
| **价值链整合型** | 砍掉中间层或合并节点 | 资本投入大 |
| **模式创新型** | 改变收费方式 (如订阅替代买断) | 用户习惯阻力 |
| **渠道创新型** | 新的触点/履约方式 | 流量成本 |
| **政策套利型** | 抓住政策窗口 | 政策风险 |

### 6.3 入局策略 6 问

对最佳机会点, 严格回答 6 问:

1. **做什么产品/服务?**
   - MVP 的核心功能是什么?
   - 一句话价值主张
   - 为什么用户会放弃现有方案切换过来?

2. **卖给谁?**
   - 细分客群 (不能是"所有人")
   - 客户画像 (人口学+心理)
   - 早期采用者特征 (参考摩尔《跨越鸿沟》)
   - 获客渠道

3. **商业模式?**
   - 收费方式 (一次性/SaaS 订阅/交易抽佣/广告/增值服务)
   - 收入流结构
   - 单位经济模型 (CAC/LTV/回本周期)
   - 关键指标

4. **如何定价?**
   - 定价锚点
   - 价格分层
   - 调价机制
   - 心理定价技巧

5. **市场有多大?**
   - TAM / SAM / SOM 三层
   - 渗透率假设
   - 5 年天花板

6. **竞争与合作** (详见 6.4 / 6.5)

### 6.4 竞争对手分析 (3 类)

| 类型 | 例子 | 应对 |
|---|---|---|
| **直接竞品** | 做同样产品同样客群 | 差异化定位 |
| **间接竞品** | 用户的钱花在别处 | 转化叙事 |
| **潜在竞品** | 巨头跨界、颠覆式新进入者 | 速度+生态 |

**每个对手写 3 件**:
- 优势 (不可撼动的)
- 劣势 (用户反馈的)
- 我的相对位置 (差异化)

### 6.5 合作伙伴分析

合作伙伴 = 放大你能力的杠杆。问 4 件事:

- **谁?**
  - 互补品厂商
  - 渠道/分销
  - 流量/平台
  - 政策资源
  - 资本/产业基金
- **合作模式?**
  - 联名/分销/转介
  - 战略投资/合资
  - 长期协议 vs 项目制
- **利益分配?**
  - 谁拿大头?
  - 谁承担风险?
  - 退出机制?
- **可替代性?**
  - 这家合作方是否唯一?
  - 议价权谁更强?

## 7. 常用理论速查

| 理论 | 核心 | 适用场景 |
|---|---|---|
| 五力模型 (Porter) | 供应商议价、买方议价、新进入者、替代品、行业内竞争 | 评估行业吸引力 |
| 价值链 (Porter) | 主要活动+支持活动 | 找价值环节 |
| 微笑曲线 (施振荣) | 两端高中间低 | 利润分布画法 |
| 4P 营销 (McCarthy) | Product/Price/Promotion/Place | 营销策略 |
| 4C 营销 (Lauterborn) | Customer/Cost/Convenience/Communication | 用户视角 |
| 4R 营销 (Schultz) | Relevance/Reaction/Relationship/Reward | 关系营销 |
| 颠覆式创新 (Christensen) | 低端/新市场颠覆 | 新进入者分析 |
| 跨越鸿沟 (Moore) | Early Adopter→Early Majority | 增长策略 |
| 定位 (Ries/Trout) | 用户心智占位 | 品牌策略 |
| 商业模式画布 (Osterwalder) | 9 大模块 | 商业模式设计 |
| VRIN (Barney) | 资源基础观 | 护城河分析 |
| 价值网 (Christensen) | 上游+下游+替代 | 商业决策 |
| 长尾 (Anderson) | 头部+无数小众 | 平台/电商 |
| 网络效应 (Metcalfe) | 用户越多价值越大 | 平台型生意 |
| 双边市场 (Rochet/Tirole) | 平台型生意 | 平台经济 |

## 8. 推演质量自检

完成推演后, 自问 5 个问题:

1. **不偏袒**: 我有没有对某个机会过度乐观? 列出反例。
2. **可证伪**: 我的判断能用什么数据/事件证伪? 如果不能, 改成假设。
3. **可执行**: 第 6 问的答案, 第一周能做什么动作?
4. **有反方**: 我有没有列出"为什么这事可能做不成"的 3 个理由?
5. **有护城河**: 如果我做了, 别人模仿我需要多久? 3 个月? 3 年? 不可模仿?

回答完整后, 才能进入输出阶段。
