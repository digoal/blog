# Impact Analysis Framework

Step 5 影响推演的方法论手册。**不要在主流程里整篇加载**, 只在写到对应主题时按需读相关小节。

## 0. 通用 4 段式输出

每条影响推演必须按下面 4 段写, 缺一段视为不完整:

| 段落 | 必须包含 | 字数建议 |
|---|---|---|
| 传导链 | 新闻事件 → 中间变量 → 行业基本面 → 上市公司, 至少 3 跳, 用箭头串成单链 | 60~120 字 |
| 理论依据 | 引用 1~2 个经济学/金融学原理 (如 DCF、利率弹性、库存周期) | 40~80 字 |
| 历史镜像 | 1 个相似历史事件 + 当时市场表现数据 | 60~120 字 |
| 受益/承压 | 3~5 个具体 ticker 或公司名, 注明交易所 | 列表 |

> **关键约束**: 受益/承压标的必须是公开可查的真实公司, 不确定就改用「行业方向」描述 (例: 「光模块龙头 800G 出货占比提升」), 严禁编造 ticker。

## 1. 8 大主题的传导链模板

### 1.1 中美关系

- **关税上调**: 出口商品价格 → 美国终端零售价 → 需求弹性 → 出口企业订单 / 毛利率
  - 理论: 微观经济学需求弹性、汇率转嫁能力
  - 历史镜像: 2018~2019 中美贸易战, A 股纺织 / 家电板块对美出口收入下滑 5~15%
- **出口管制 (实体清单)**: 上游材料 / 设备断供 → 国产替代窗口打开 → 国内供应商市占率
  - 理论: 波特五力 (供应商议价能力骤降)、产业政策红利
  - 历史镜像: 2019 华为被列入实体清单, 国产模拟 / 存储公司后续 3 年市占率提升 5~10pct
- **典型标的**: 半导体设备 (北方华创、中微公司)、模拟芯片 (圣邦股份、纳芯微)、跨境电商 (Temu、SHEIN 母公司)

### 1.2 地缘局势

- **中东冲突升级**: 原油运输风险溢价 → 油价 → 全球通胀 / 美联储路径 / 美元
  - 理论: 大宗商品定价中的地缘溢价、滞胀逻辑
  - 历史镜像: 2022 俄乌冲突后 3 个月内 Brent 油价从 80 冲高至 130 USD
- **关键航运咽喉 (红海 / 霍尔木兹) 中断**: 集装箱 / 油轮绕行 → 运价 → 航运公司业绩
  - 理论: 供给冲击经济学
  - 历史镜像: 2023 底红海危机, SCFI 集装箱运价指数短期翻倍
- **典型标的**: 中海油、中远海控、中国船舶; 黄金股 (山东黄金、紫金矿业); 军工 (航发动力、中航西飞)

### 1.3 美联储

- **降息预期升温**: 美元 ↓ 美债收益率 ↓ 久期长的成长股 / 小盘股 / 生物科技 ↑
  - 理论: DCF 分母下降、流动性溢价收缩
  - 历史镜像: 2019 预防式降息周期, NASDAQ 100 半年内 +20%, XBI 生物科技 ETF 同期 +30%
- **点阵图鹰派转向**: 新兴市场资本外流压力 ↑, 人民币贬值压力, A 股北向资金流出
  - 理论: 利差驱动的国际资本流动
  - 历史镜像: 2022 鹰派加息, MSCI 中国全年 -21%, 沪深 300 -22%
- **典型标的**: 美股 7 姐妹 (NVDA、MSFT、GOOGL、META、AAPL、AMZN、TSLA); 黄金 ETF (GLD); 港股流动性敏感板块 (恒生科技); A 股高股息红利 (长江电力、中国神华)

### 1.4 民生数据

- **CPI 超预期上行**: 必选消费提价空间 ↑, 利率上调预期 ↑, 估值压力
  - 理论: 通胀预期与定价权传导
  - 历史镜像: 2007 通胀周期, 食品饮料板块相对收益领先大盘 15pct
- **地产销售持续走弱**: 房企现金流 → 上游建材 / 中游家电 / 下游物业 全链条承压
  - 理论: 财富效应、上下游信用收缩
  - 历史镜像: 2021~2023 地产链, 申万房地产指数累计 -40%, 家电 -30%
- **典型标的**: 白酒 (贵州茅台、五粮液)、乳制品 (伊利)、家电 (美的、格力)、猪企 (牧原)、地产链建材 (东方雨虹、海螺水泥)

### 1.5 高科技

- **存储芯片涨价周期**: DRAM/NAND 现货价 ↑ → 模组厂毛利率扩张 → 国产存储企业
  - 理论: 半导体周期 (硅周期), 通常 2~3 年一轮
  - 历史镜像: 2023Q3~2024Q2 存储涨价周期, 三星电子营业利润同比 +10 倍, 兆易创新涨幅 +80%
- **AI 算力需求 → 先进制程产能**: 台积电 / 三星排产紧 → 国产代工溢出效应
  - 理论: 产能稀缺溢价、产业链外溢
  - 历史镜像: 2024 CoWoS 产能紧张, 台积电 CoWoS 报价上调 15~25%
- **典型标的**: 存储 (兆易创新、长鑫存储-未上市)、封测 (长电科技、通富微电)、设备 (北方华创、中微)、光刻胶 (南大光电)

### 1.6 生物医疗

- **创新药 License-out 大单**: 首付款 + 里程碑收入 → 估值切换 (从 PE 转向管线 NPV)
  - 理论: 风险调整后净现值 (rNPV) 估值
  - 历史镜像: 2023 末~2024 康方生物 AK112 5 亿美金首付款, 公司估值短期 +60%
- **集采降幅超预期**: 仿制药 / 器械毛利率压缩, 倒逼企业转型创新
  - 理论: 政府采购的价格谈判机制、医保战略性购买
  - 历史镜像: 2020 冠脉支架集采降幅 93%, 乐普医疗短期 -30%
- **GLP-1 减肥药放量**: 司美格鲁肽 / 替尔泊肽 → 原料药 CDMO 弹性最大
  - 理论: 产能转移 + 原料药杠杆
  - 历史镜像: 2023 GLP-1 类药物全球销售额破 200 亿美金, 国内多肽 CDMO 订单激增
- **典型标的**: 创新药 (百济神州、信达生物、康方生物)、CRO (药明康德、康龙化成)、多肽 CDMO (诺泰生物、圣诺生物)

### 1.7 云计算

- **Hyperscaler capex 上调**: 光模块 / IDC / 液冷需求 → 800G/1.6T 出货加速
  - 理论: 资本开支 → 上游元器件订单传导
  - 历史镜像: 2024 全年 4 大云厂 capex 合计超 2000 亿美金, 中际旭创、新易盛年内 +200%
- **云厂商价格战**: IaaS 毛利率 ↓, 但应用层 SaaS 受益 (客户基础扩大)
  - 理论: 平台经济学、长尾需求
  - 历史镜像: 2020~2021 国内云大降价, 金山办公 / 用友等 SaaS 渗透率提升
- **典型标的**: 海外 (MSFT、GOOGL、AMZN); A 股光模块 (中际旭创、新易盛、天孚通信); IDC (润泽科技、奥飞数据); 液冷 (英维克、申菱环境)

### 1.8 AI 产业链

- **英伟达季度业绩超预期 + 指引上修**: GPU 供应链 → HBM / 先进封装 / CCL / 电源订单
  - 理论: 产业链订单与排产
  - 历史镜像: 2024 年至今英伟达每季报后 1 个月内, A 股光模块 / 液冷板块跟随度 0.7+
- **国产 GPU 突破**: 模型训练成本 ↓, 国内大模型迭代加速 → 算力 / 应用齐涨
  - 理论: 技术扩散与生态外溢
  - 历史镜像: 2024 华为昇腾 910B / 寒武纪思元 590 出货, 国产替代率从 30% 升至 50%
- **典型标的**: 算力 (寒武纪、海光信息、龙芯中科); HBM (香农芯创、雅克科技); 端侧 AI (恒玄科技、中科蓝讯); AI 应用 (金山办公、科大讯飞)

## 3. 推演质量自检 (写完一段后默念)

- [ ] 传导链 ≥ 3 跳, 箭头连贯, 没有跳跃
- [ ] 理论依据点了具体原理名 (DCF / 弹性 / 库存周期 / 汇率传导)
- [ ] 历史镜像给了具体年份 + 数字
- [ ] 受益/承压是真实 ticker 或明确行业方向, 没有编造
- [ ] 区分了短期情绪 (1~2 周) 和中期基本面 (1~3 个季度) 影响

任一条不满足, 重写该段。
