# Topics & Search Keywords

8 个默认主题。每个主题给出: 主题说明 / 中英文搜索关键词 / 关键事件类型 / 关注标的(举例, 实时性需二次确认)。

> **使用提示**: Step 1 抓取时, 把 `中文 query` 和 `English query` 至少各发一次, 提高召回。中美关系 + 地缘局势经常有重叠, 抓到的事件先去重再归类。

---

## 1. 中美关系 (US-China Relations)

- **中文 query**: `中美关系 最新`、`中美贸易 关税`、`TikTok 禁令`、`中美科技 出口管制`
- **English query**: `US-China relations latest`、`US-China tariff`、`CHIPS Act`、`entity list China`
- **关键事件类型**:
  - 关税 / 制裁 / 出口管制清单调整
  - 大厂在美 IPO / 退市 / 审计底稿问题
  - 高层通话、APEC / G20 外交
  - TikTok、微信、华为等具体企业级事件
- **关注方向**: 半导体设备、AI 算力、消费电子出海、新能源车出口、跨境电商

## 2. 地缘局势 (Geopolitics)

- **中文 query**: `俄乌战争 最新`、`中东局势 油价`、`台海 紧张`、`朝鲜 导弹`
- **English query**: `Russia Ukraine war latest`、`Middle East oil supply`、`Taiwan strait tension`、`Red Sea shipping`
- **关键事件类型**:
  - 主要冲突区升级 / 降级
  - 能源 / 粮食 / 关键矿产供应链中断
  - 红海 / 霍尔木兹 / 巴拿马运河等咽喉航运
  - 关键国家选举 / 政府更迭
- **关注方向**: 原油、黄金、航运、化肥、军工、稀土

## 3. 美联储 (Federal Reserve)

- **中文 query**: `美联储 加息 降息`、`FOMC 会议纪要`、`鲍威尔 讲话`、`美国 CPI 非农`
- **English query**: `Fed rate decision`、`FOMC minutes`、`Powell speech`、`US CPI nonfarm payrolls`
- **关键事件类型**:
  - FOMC 决议与点阵图
  - 鲍威尔 / 威廉斯 / 沃勒等关键票委讲话
  - CPI / PCE / 非农 / 失业率
  - 10Y / 2Y 美债收益率倒挂变化
  - 美元指数 DXY 剧烈波动
- **关注方向**: 全球流动性、成长股久期、黄金、生物科技、新兴市场债、REITs

## 4. 民生数据 (Livelihood / Macro Demand)

- **中文 query**: `中国 CPI PPI`、`社零 消费`、`房地产 销售`、`PMI 制造业`
- **English query**: `China CPI PPI`、`China retail sales`、`China property sales`、`China PMI manufacturing`
- **关键事件类型**:
  - 月度 CPI / PPI / 社零 / 工业增加值
  - 70 城房价、地产销售面积
  - 居民贷款 / 存款数据
  - 失业率、应届生就业
  - 春节 / 618 / 双 11 / 黑五等节点消费数据
- **关注方向**: 食品饮料、家电、社服、零售、地产链、保险

## 5. 高科技 (High-Tech)

- **中文 query**: `半导体 国产替代`、`光刻机 ASML`、`存储芯片 涨价`、`HBM 内存`
- **English query**: `semiconductor China`、`ASML EUV lithography`、`DRAM NAND price`、`HBM3 HBM4 memory`
- **关键事件类型**:
  - 半导体设备 / 材料 / 制造产能与良率
  - 存储 / 逻辑 / 模拟各类芯片价格周期
  - AI 芯片 (GPU / ASIC / TPU) 出货与代工排产
  - 苹果 / 华为 / 三星旗舰机型与产业链
  - 卫星互联网、机器人、量子计算
- **关注方向**: 半导体设备、材料、设计、代工、封测、消费电子、果链

## 6. 生物医疗 (Biomedical)

- **中文 query**: `创新药 医保谈判`、`GLP-1 减肥药`、`ADC 药物 出海`、`医疗器械 集采`
- **English query**: `innovative drug China reimbursement`、`GLP-1 obesity drug`、`ADC licensing deal`、`medical device VBP`
- **关键事件类型**:
  - FDA / NMPA 关键审评决议 (PDUFA 日期)
  - 医保谈判、药品集采结果
  - 国产创新药海外授权 (License-out) 大单
  - 跨国大药企 (MNC) 财报与指引
  - 流感 / RSV / 新发传染病监测
- **关注方向**: 创新药企 (恒瑞、百济、信达)、CRO/CDMO、减肥药产业链、医美、血制品

## 7. 云计算产业 (Cloud Computing)

- **中文 query**: `云厂商 资本开支`、`AWS 财报`、`Azure 增速`、`阿里云 降价`
- **English query**: `hyperscaler capex`、`AWS earnings`、`Azure growth`、`Alibaba cloud price cut`
- **关键事件类型**:
  - AWS / Azure / GCP / 阿里云 / 腾讯云季度财报与 capex 指引
  - 公有云市场份额变化 (Synergy / Canalys)
  - 大模型 API 价格战、token 调用量
  - 数据中心建设、电力供应、光模块需求
  - SaaS 公司业绩与 ARR
- **关注方向**: 公有云厂商、光模块 (800G/1.6T)、IDC、CDU 液冷、SaaS 头部公司

## 8. AI 产业链 (AI Supply Chain)

- **中文 query**: `英伟达 财报`、`国产 GPU 进展`、`DeepSeek 开源`、`AI 智能体 agent`
- **English query**: `NVIDIA earnings`、`China AI chip progress`、`DeepSeek open source`、`AI agent launch`
- **关键事件类型**:
  - 英伟达 / AMD / 博通 AI 芯片出货与客户结构
  - 国产 GPU / ASIC (昇腾、寒武纪、海光、摩尔线程) 进展
  - 头部模型 (GPT、Claude、Gemini、Llama、DeepSeek) 发布与基准
  - AI Agent / Copilot / 端侧 AI 商业化进展
  - AI 电力消耗、核电重启、数据中心选址
- **关注方向**: GPU 设计与代工、HBM / 先进封装、CCL / PCB、液冷、电力运营商

## 用户自定义行业

用户传 1 个行业名时, 按下面模板拆解:

1. **核心 query**: `<行业中文> 最新 新闻`、`<industry English> latest news`
2. **上下游 query**: 上游原材料 / 下游应用 各 1 个
3. **大厂财报 / 政策 / 技术** 至少各覆盖一条

> 提醒: 自定义行业要保持「外科手术式」克制, 不要自动扩展到不相关主题。
