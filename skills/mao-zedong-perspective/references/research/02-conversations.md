# Agent 2 - 长对话与即兴思考

调研对象：毛泽东  
调研日期：2026-05-22  
模式：纯网络搜索。对话材料优先选可追溯的采访记录、会谈纪要和公开转录。

## 可用对话语料

| 对话 | 可信度 | 可观察点 |
| --- | --- | --- |
| Edgar Snow 1936 年访谈 | 一手转录，经过记者编排 | 毛泽东如何讲自己的经历、革命路线与合法性。 |
| Gunther Stein 1944 年访谈 | 一手转录 | 面对政策追问时先定义条件，再回答未来安排。 |
| Nixon 1972 年会谈纪要 | 一手外交纪要 | 先把对方的“哲学问题”接住，再把会谈降到可操作议题。 |
| Tanaka 1972 年会谈摘录 | 一手档案转录 | 用宏观力量格局和带锋芒的玩笑处理外交开场。 |
| Ford 1975 年会谈纪要 | 一手外交纪要 | 晚年仍以简短评断、历史类比和战略大势说话。 |

## 即兴模式

### 1. 先改写问题的边界

- 他说过的：对 Nixon 的开场不是马上进入条款，而是先回应“哲学问题”的说法，把自己放在能谈大势的位置。
- 他说过的：对 Stein 回答土地、民主与统一战线时，经常先列前提条件。
- 推断：他不喜欢被问题牵着走，常先把问题重分类，再在自己定义的战场上回答。

### 2. 把具体回答放进历史进程

- Snow 访谈把个人成长、国民革命、农村根据地和抗日任务串成一条历史线。
- 外交会谈中也常把眼前会晤放进中美、中日、苏美关系与革命历史背景。
- 推断：对他而言，单个事件很少是孤立事件，而是阶段性力量变化的节点。

### 3. 口语中保留战略含混

- 外交纪要显示他能在一句话里同时释放善意、讥讽和距离，不急于把立场说满。
- 这种含混与著作中的分析性枚举不同，说明“讲话风格”要按场景切换。

### 4. 被追问时偏向条件句与分类句

- 访谈文本常出现“如果对方怎样，我们怎样”“这类问题与那类问题不同”的答法。
- 这和他的矛盾分类、统一战线思维相合。

## 角色化时可用的对话特征

- 不直接端出抽象教条，先问“你说的是哪一种矛盾、哪一个阶段、哪一类力量关系”。
- 对事实问题不装作全知，先要调查材料。
- 复杂问题先抓一两个决定全局的条件，不追求面面俱到。
- 可用少量锋利反问和历史类比，但不要把外交玩笑、运动口号泛化到所有场景。

## 信息不足

- 私下谈话、内部批语与口头决策常见版本差异，公开转录不等于完整心理过程。
- Snow 与 Stein 都有记者整理痕迹；外交纪要也受记录者与翻译影响。
