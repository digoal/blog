# Agent 5 - 决策记录与行动

调研对象：毛泽东  
调研日期：2026-05-22  
模式：纯网络搜索，决策案例同时记录有效方法与失效边界。

## 决策案例

| 案例 | 可见逻辑 | 结果与风险 |
| --- | --- | --- |
| 1927 年湖南农民运动调查 | 把革命主体判断建立在实地观察，而不是城市中心教条 | 形成农村革命与农民动员的路线判断。 |
| 井冈山、根据地与游击/运动战路线 | 承认敌强我弱，用空间、群众、机动和时间重排力量 | 成为弱势条件下的战略创新。 |
| 遵义后军事路线调整 | 反对脱离中国战场条件的僵硬指挥 | 其后在党内形成更强领导地位。 |
| 抗日统一战线与持久战 | 把当前主要矛盾与力量阶段放在中心 | 说明他能为阶段目标重排联盟。 |
| 1949-1950 与苏联谈判 | 在意识形态同盟中仍争取国家利益与安全支持 | 展示战略耐心和大国周旋。 |
| 1956-1957 百花与反右转折 | 先释放批评，再将部分批评重判为敌对 | 暴露矛盾分类扩张后的政治伤害。 |
| 1958 大跃进与人民公社 | 把群众动员、意志与跃迁期待推向经济治理 | 事实反馈被扭曲，造成严重损失。 |
| 1959 庐山后反右倾 | 面对纠偏信号，政治忠诚判断压过政策复盘 | 反证“调查研究”在高压组织中会失效。 |
| 1966 发动文革 | 用社会动员冲击党内官僚化与路线对手 | 造成长期秩序破坏和巨大人身代价。 |
| 1972 中美接近 | 在中苏裂变与国际格局中重新排序主要威胁 | 说明主要矛盾判断也可导向灵活外交。 |

## 提炼出的决策启发式候选

1. 先调查对象与条件，再谈路线。
2. 先找当前决定全局的主要矛盾。
3. 弱势时不硬拼既定规则，换时间尺度、组织方式和作战形态。
4. 把群众的经验变成领导判断，再回到行动里检验。
5. 阶段变化时联盟也要变化。
6. 领导要抓典型点，不能只靠一般号召。
7. 不把政治动员当作替代物质、专业和反馈的万能杠杆。
8. 不能把批评、分歧和敌对混成一类。

## 言行一致与不一致

### 一致

- 早期反教条文本与农村调查、游击战略、持久战判断一致。
- 对官僚化与革命蜕化的警惕贯穿延安领导方法、建国后路线分歧和文革发动动机。

### 不一致或失效

- “实践检验”与大跃进时期坏消息上行受阻形成强反差。
- “人民内部矛盾分类处理”与反右、文革中的扩大化存在强反差。
- “从群众中来，到群众中去”在后期有时退化成从上而下发动群众。
