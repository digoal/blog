# Agent 6 - 人物时间线

调研对象：毛泽东  
调研日期：2026-05-22  
模式：历史人物时间线，最近动态项改为“材料稳定性与后世评价节点”。

## 关键时间线

| 时间 | 节点 | 对思维画像的意义 |
| --- | --- | --- |
| 1893 | 生于湖南韶山 | 农村中国经验成为后续政治想象的重要底色。 |
| 1918 | 湖南第一师范毕业 | 教育、阅读、组织与新文化冲击汇合。 |
| 1921 | 参与中共创建时期活动 | 进入革命组织政治。 |
| 1927 | 写湖南农民运动调查报告 | 农村调查与农民动员路线成形。 |
| 1927-1934 | 井冈山与中央苏区阶段 | 根据地、游击与农村包围城市经验沉淀。 |
| 1935 | 长征途中遵义会议后地位上升 | 战略自主与党内领导权同步强化。 |
| 1936-1938 | 战略与哲学文本密集形成 | 战争、实践、矛盾分析成为体系。 |
| 1942-1943 | 延安整风、文艺讲话、领导方法文本 | 思想统一、群众路线与政治文化关系被制度化。 |
| 1949 | 宣告中华人民共和国成立 | 革命领导者转入国家治理者。 |
| 1950 | 与苏联缔结条约 | 革命外交进入国家安全与发展约束。 |
| 1956-1957 | 百花、正确处理人民内部矛盾、反右转折 | 分类矛盾与政治风险判断发生高代价碰撞。 |
| 1958-1961 | 大跃进与困难时期 | 群众动员、赶超冲动与反馈失真成为后期风险核心。 |
| 1966 | 发动文化大革命 | 反官僚化与路线斗争走向社会动员冲击秩序。 |
| 1972 | 会见 Nixon、推动中美接近 | 主要矛盾判断在国际格局中体现出灵活性。 |
| 1976 | 去世 | 生前政治路线告一段落。 |
| 1981 | 中共通过建国以来历史问题决议 | 后世官方评价明确区分贡献与严重错误。 |

## 稳定性说明

- 毛泽东已于 1976 年去世，不存在“最近 12 个月本人动态”。
- 史料仍会因档案开放、版本校勘与研究解释变化而变化；Skill 调研截止日仍需保留。
