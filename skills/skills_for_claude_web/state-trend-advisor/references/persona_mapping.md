## 六、普通人 / 无特定背景

**若询问者无特定专业背景，优先推荐以下"普通人友好型"趋势：**

1. **新能源相关服务**：充电桩运营、光伏安装维护（政策补贴+低技术门槛）
2. **银发经济服务**：家政、养老陪护、老年教育（人口结构驱动，进入门槛低）
3. **乡村振兴**：农村电商、特色农产品、乡村文旅（政策资金+补贴齐备）
4. **职业技能培训**：考证培训、技能证书（政策补贴，稳定需求）
5. **跨境电商运营**：选品、运营（政策支持出口，工具成熟）

## 八、历史经验库（用于报告中的类比支撑）

| 年份 | 政策事件 | 结果 | 经验教训 |
|------|---------|------|----------|
| 2012-2015 | 4G牌照发放 | 移动互联网十年黄金期 | 基础设施政策领先市场3-5年 |
| 2014-2016 | 双创政策+互联网+ | 创业潮，千团大战 | 风口期竞争烈，后发者需差异化 |
| 2015-2020 | 新能源汽车补贴 | 比亚迪等崛起 | 早期补贴期是布局窗口 |
| 2017-2020 | 直播电商兴起 | 李佳琦、薇娅现象 | 内容政策+消费升级叠加效应 |
| 2020-2022 | 医疗集采深化 | 国产器械取代进口 | 强监管重塑竞争格局时是机会 |
| 2021-2023 | 碳中和政策落地 | 光伏储能爆发 | 政策确定性高时，产业链整体受益 |
| 2023-2025 | AI大模型政策支持 | 百模大战，应用爆发 | 平台期过后应用层机会更大 |
