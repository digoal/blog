# 美股个股次日走势预测方法论

## 一、核心哲学

**本质认知：** 股价短期走势 = 信息流 × 资金流 × 情绪流 的综合博弈结果。
次日预测不追求"精确点位"，而是判断**方向概率**与**风险收益比**，以此指导是否持有/加仓/减仓/做空。

**三不预测原则：**
1. 不预测黑天鹅（无法预测的不预测）
2. 不给出绝对价格（只给概率区间）
3. 不预测震荡行情（信号不明时明确说"中性/观望"）

---

## 二、五维分析框架（MTVSF Model）

### M - Macro Pulse（宏观脉搏）
权重：15%

宏观环境决定市场整体β值，个股在其中浮沉。

**检查项：**
- 美联储最新态度（鹰/鸽）
- 10年期美债收益率变动（>5bp 视为显著）
- VIX 恐惧指数水平（<15 贪婪区，15-25 中性，>25 恐惧区，>30 极度恐慌）
- 美元指数（DXY）方向（强美元压制成长股）
- 标普500/纳斯达克当日走势与次日期货

**评分逻辑：**
- 宏观顺风 → +0.5 到 +1.5 分（满分15分制）
- 宏观逆风 → -1.5 到 -0.5 分

---

### T - Technical Signals（技术信号）
权重：30%

技术分析反映市场参与者的集体心理预期，短期最有效。

**核心指标体系：**

#### 1. 趋势判断（必查）
- 日线收盘价 vs 20日/50日/200日均线（MA）
  - 价格 > MA200 = 长期多头，> MA50 = 中期多头
  - 黄金交叉（MA50上穿MA200）= 强势信号
- 布林带（Bollinger Bands）：收盘在上轨附近=超买压力，下轨=支撑位

#### 2. 动量指标（必查）
- RSI（相对强弱指数）：
  - > 70 = 超买（警惕回调）
  - < 30 = 超卖（关注反弹）
  - 50附近 = 中性
- MACD（异同移动平均线）：
  - 金叉（MACD线上穿信号线）= 买入信号
  - 死叉 = 卖出信号
  - 柱状图扩大方向 = 动能方向

#### 3. 成交量分析（关键）
- 量价关系：
  - 放量上涨 = 强势确认（多头力量强）
  - 缩量下跌 = 弱势调整（可能企稳）
  - 放量下跌 = 危险信号（空头主导）
  - 缩量上涨 = 虚涨（可疑，后续需确认）
- OBV（能量潮指标）：OBV持续上升 = 筹码积累中

#### 4. 支撑/阻力位（必查）
- 前期高低点
- 整数关口（心理价位）
- 历史成交密集区
- Fibonacci回撤位（38.2%, 50%, 61.8%）

**技术信号综合评分：** 3个以上同向信号 = 强信号；1-2个同向 = 弱信号；信号矛盾 = 中性

---

### V - Valuation & Fundamentals（估值与基本面）
权重：20%

基本面是价值锚，短期可能偏离，但极端偏离会触发均值回归。

**检查项：**
- P/E 相对行业均值（>2倍行业均值 = 高估风险）
- 近期盈利修正方向（分析师上调/下调EPS预测）
- 距下次财报日期（财报前5日波动率往往放大）
- PEG（P/E÷增长率）：< 1 = 低估，> 2 = 高估
- 机构评级变化（升级/降级）

**特别注意：** 财报日前后是最重要的催化剂窗口。

---

### S - Sentiment & News Flow（情绪与消息流）
权重：25%

市场短期由情绪驱动，消息是情绪的触发器。

**检查项：**

#### 新闻质量分级
- 🔴 高影响：财报超预期/不及预期、并购、FDA审批、重大诉讼、CEO离职
- 🟡 中影响：分析师评级变化、产品发布、合同获得、管理层发言
- 🟢 低影响：行业报道、次要分析师评论

#### 情绪量化工具
- Put/Call Ratio（期权市场情绪）：
  - > 1.2 = 市场偏空（但可能是超卖信号）
  - < 0.6 = 市场偏多（但可能超买）
- 短卖比例（Short Interest）：
  - Short Float > 20% = 存在逼空（Short Squeeze）潜力
  - 空头回补 + 上涨 = 强烈信号
- 社交媒体情绪（Reddit/X平台）：散户情绪的领先指标

#### 新闻时效性原则
- 今日盘后/明日盘前消息 > 今日盘中消息 > 昨日消息
- 已被市场充分消化的消息 = 中性（不再是催化剂）

---

### F - Flow & Positioning（资金流与持仓）
权重：10%

"聪明钱"的动向是重要参考。

**检查项：**
- 当日资金净流入/流出
- 大宗交易（Block Trade）异动
- 期权异常成交（Unusual Options Activity）：
  - 大量买入看涨期权（Call）= 聪明钱押注上涨
  - 大量买入看跌期权（Put）= 聪明钱对冲或押注下跌
- 机构13F持仓变化（季度更新）
- ETF资金流（相关ETF的资金流代表行业趋势）

---

## 三、综合评分系统（100分制）

```
总分 = M得分(0-15) + T得分(0-30) + V得分(0-20) + S得分(0-25) + F得分(0-10)
```

**分数解读：**
| 总分区间 | 方向判断 | 建议操作 | 置信度 |
|---------|---------|---------|--------|
| 75-100  | 强烈看涨 | 可考虑买入/加仓 | 高 |
| 60-74   | 温和看涨 | 轻仓介入，设止损 | 中 |
| 45-59   | 中性/震荡 | 观望为主 | 低 |
| 30-44   | 温和看跌 | 减仓/谨慎持有 | 中 |
| 0-29    | 强烈看跌 | 回避或做空 | 高 |

## 四、概率化输出模型

### 情景分析（三情景法）

对于每只股票，必须输出三种情景：

**情景A：看涨情景**（明日涨幅 > +1%）
- 触发条件描述
- 概率估计：P(A)%

**情景B：震荡情景**（明日涨跌在 -1% 到 +1%）
- 触发条件描述
- 概率估计：P(B)%

**情景C：看跌情景**（明日跌幅 > -1%）
- 触发条件描述
- 概率估计：P(C)%

*注：P(A) + P(B) + P(C) = 100%*

### 预期收益计算

```
预期涨幅 = P(A) × 看涨情景涨幅 + P(B) × 0 + P(C) × 看跌情景跌幅
```

如果预期涨幅 > 0 且 风险收益比 > 1.5，则具有操作价值。

## 五、风险控制铁律

1. **止损线必须先于入场设置**：通常设在关键支撑位下方 2-3%
2. **单笔仓位不超过总资金 5%**（防止黑天鹅）
3. **财报前夕减半仓位**（财报是最大不确定性）
4. **VIX > 30 时全面保守**（市场极端恐慌，预测失效率大幅上升）
5. **连续3次预测失误后停止交易，复盘方法论**

## 七、方法论局限性声明

- 本方法论基于历史规律，不保证未来有效
- 次日预测准确率行业标准约55-65%（随机猜测50%）
- 以下情况方法论失效：
  - 重大突发新闻（黑天鹅）
  - 市场流动性危机
  - 监管层突然干预
  - 算法程序化异常交易
