# State Trend Advisor（国势趋势顾问）

> "人应该顺势而为，跟着国家走，走在趋势上，赚势头的钱。"

## Overview

本 skill 帮助用户把国家战略转化为个人/组织决策。它抓取并分析：
- 国家最新五年规划（如：中国 十五五规划 2026-2030）
- 重大政策文件（国务院/部委文件、白皮书、行动计划）
- 近期重大新闻（顶层会议、立法、重大工程、突发事件）

然后面向六大场景输出报告：
1. **投资**（资产配置、行业选择、择时择势）
2. **创业方向**（赛道选择、模式选择、风险识别）
3. **商业思路**（存量生意的转型升级、新业务孵化）
4. **产品思路**（产品定位、政策红利、用户痛点结合）
5. **就业择业**（行业景气度、岗位缺口、城市选择）
6. **学习规划**（专业选择、技能学习、考证考研）

## When to Use This Skill

- 用户说"分析国家政策方向"、"顺着国家走"、"国家现在主推什么"
- 用户提问"现在适合投什么 / 创什么 / 学什么 / 选什么行业"
- 用户给出个人画像（特长/爱好/专业）想看自己与国家趋势的契合度
- 用户想看五年规划、十四五/十五五、政策红利的解读
- 用户做年度复盘或 3-5 年规划，希望以国家趋势为锚

不适用：纯技术问题、与国家政策无关的纯私人决策、纯娱乐/闲聊。

## Core Philosophy

### 势—道—术 三层推演

| 层级 | 内容 | 推演依据 | 时间窗 |
|---|---|---|---|
| **势** | 国家想做什么、压什么、放什么 | 五年规划 + 党/国务院文件 + 重大会议 | 5-10 年 |
| **道** | 产业链如何重塑、钱从哪流向哪 | 财政/货币/产业/区域/科技/民生/对外 6 类政策 | 3-5 年 |
| **术** | 个人/组织在 6 大场景的具体动作 | 行业景气数据 + 岗位数据 + 案例 | 1-3 年 |

> 类比：势是河流的方向，道是水流冲出的河道，术是你在哪个河段、用什么姿势下水。
> 推演原则：先看势（不可抗力）→ 再看道（结构变化）→ 最后看术（可操作机会）。
> 推演准则：先看国家明确支持的"鼓励类"，再看"限制类"边界，最后看"默许类"灰度机会。

### 顺势 ≠ 盲从

- **顺势**：识别趋势 → 顺势布局 → 享受红利
- **盲从**：政策一出一拥而上 → 政策退坡一地鸡毛
- 推演时要标注政策周期阶段（导入/成长/成熟/退坡），避免在退坡期入场

## Workflow

按以下顺序执行。每步都有 🛑 强制 Checkpoint，未完成前不进入下一步。

### 🛑 Checkpoint 0：用户输入确认

**必须** 在动手抓数据前向用户确认以下信息（一次性问完，不要分轮次问）：

1. **目标国家/地区**：默认"中国"，可指定美国/欧盟/日韩/其他
2. **目标场景**：单选或多选 — 投资 / 创业方向 / 商业思路 / 产品思路 / 就业择业 / 学习规划
3. **用户画像**：
   - 擅长领域（如：AI、电商、医疗、制造业）
   - 特长/技能（如：算法、销售、内容、设计）
   - 爱好（如：二次元、户外、阅读、烹饪）
   - 现有资源（如：行业经验、渠道、资金、人脉）
   - 城市/地域（影响行业聚集与政策试点）
4. **报告风格**：综合型（6 场景全覆盖）vs 专项型（深挖 1-2 个场景）
5. **报告长度**：长报告（5000+字，含多图）vs 短报告（2000-3000字，精简）

> 若用户未给画像，按"普通人"处理：本科以上学历、25-35岁、一二线城市、可支配资金 10-100 万、无强行业积累。

> **🛑 STOP**：未拿到完整 5 项输入前，禁止进入 Step 1 抓数据。

### Step 1：抓取国家政策与新闻

调用 `mcp__MiniMax__web_search`（**强制使用 MCP 工具，禁止使用内置 web_search**）按以下顺序抓取：

1. **顶层规划**：当前五年规划全文 + 历次全会公报（"十五五规划 全文"、"十五五规划纲要 解读"）
2. **重大政策**：近 90 天国务院常务会议、重大文件、白皮书、行动计划
3. **重大新闻**：近 30 天顶层会议（中央经济工作会议、政治局会议）、立法、重大工程、突发
4. **行业数据**：目标场景相关行业的 GDP 占比、增速、利润、投资、政策补贴
5. **对照参考**：上一轮五年规划评估报告、行业白皮书

每次搜索至少 3 个 query 维度（顶层/执行/对照），结果按时间倒序 + 来源权威度排序。

### Step 2：信号抽取与降噪

参考 [policy-signal-taxonomy.md](references/policy-signal-taxonomy.md) 把抓到的内容归入 7 类信号：
- 财政（钱从哪来/到哪去）
- 货币（钱多钱少、利率汇率）
- 产业（鼓励/限制/淘汰/试点）
- 区域（哪个城市/地区被点名）
- 科技（卡脖子/攻关/转化）
- 民生（就业/教育/医疗/住房）
- 对外（一带一路/脱钩/合作）

每类信号提取 3-5 个**可验证**事实（带数字、出处、时间），丢弃情绪化/标题党/未核实信息。

### Step 3：势—道—术 推演

参考 [trend-analysis-framework.md](references/trend-analysis-framework.md) 的 5 步推演法：

1. **定位势**：当前国家处于什么周期（导入/成长/成熟/退坡）？方向是什么？
2. **识别道**：产业链的钱/政策/人流如何重塑？谁受益、谁受损？
3. **匹配术**：参考 [user-profile-mapping.md](references/user-profile-mapping.md) 把用户画像与机会匹配
4. **类比历史**：从 [historical-case-bank.md](references/historical-case-bank.md) 找 1-3 个相似历史案例做支撑
5. **风险标注**：标注每个机会的 3 类风险（政策周期/执行落差/能力不匹配）

### Step 4：场景化输出

按用户选定的场景，参考 [scenario-templates.md](references/scenario-templates.md) 套用对应模板：
- 6 场景结构基本一致（判断→机会→风险→行动→自查）
- 投资：增配/减配/择时 + 标的（板块/ETF/个股方向，不给具体代码）
- 创业：赛道 → 模式 → 团队 → 资金 → 退出
- 商业：存量诊断 → 转型路径 → 新业务孵化
- 产品：用户痛点 → 政策红利 → 商业模式
- 就业：行业景气 → 岗位缺口 → 城市选择 → 简历方向
- 学习：专业 → 院校 → 技能 → 证书 → 路径

### Step 5：图文并茂

按需生成可视化（每份报告 ≥ 2 张图）：

| 图形 | 用途 | 工具 |
|---|---|---|
| 趋势雷达图 | 6 大场景的机会分值 | 外挂 SVG（见 [assets/svg/trend-radar.svg](assets/svg/trend-radar.svg)） |
| 时间线 | 5-10 年政策演化 | Mermaid `gantt` 或外挂 SVG |
| 机会矩阵 | 政策红利 × 用户能力 2x2 | 外挂 SVG（见 [assets/svg/opportunity-matrix.svg](assets/svg/opportunity-matrix.svg)） |
| 产业链图谱 | 钱/政策/人/技术在产业链上的流向 | Mermaid `flowchart` |
| 政策推演树 | 因果链 + 关键节点 | Mermaid `flowchart` |
| 数据对比表 | 行业历史/中外/上轮规划对比 | Markdown 表格 |

**SVG 使用规范**：
- ✅ 必须**外挂式** SVG 文件，保存到 `<markdown-dir>/svg/<report-slug>-<chart>.svg`
- ✅ markdown 中用 `![描述](svg/<report-slug>-<chart>.svg "标题")` 引用
- ❌ **禁止** 在 markdown 中嵌入 `<svg>...</svg>` 内联代码
- 提供 [assets/svg/](assets/svg/) 下的 4 个模板作为起点

### Step 6：质量检查

写完报告前必须自查（命中任一即重写）：

- [ ] 推演链有"因为国家说 A → 行业 B 受益 → 钱流向 C → 用户可做 D"完整闭环
- [ ] 至少 3 个事实带数据/出处（不是"普遍认为"）
- [ ] 至少 1 个历史案例支撑（"上次 XX 规划时类似情况是 XX"）
- [ ] 每个机会标注了风险，不是单边推荐
- [ ] 用户画像若为空，明确写"按普通人画像推演"
- [ ] ≥ 2 张图，且 SVG 都是外挂引用

### Step 7：输出到 markdown 目录

文件命名：
```
markdown/state-trend-<goal>-<date>.md
```
- `<goal>`：用户选定的场景（investment/entrepreneur/business/product/career/study/multi）
- `<date>`：`YYYYMMDD`
- 示例：`markdown/state-trend-investment-20260611.md`

SVG 文件保存到：
```
markdown/svg/<report-slug>-<chart>.svg
```
- `<report-slug>` 与同名报告去后缀
- 示例：`markdown/svg/state-trend-investment-20260611-trend-radar.svg`

## Output Format

报告必须包含以下 7 个章节（专项型报告可压缩前 2 章）：

### 1. 一句话核心判断

> 一句话讲清楚"国家当前在干什么、个人该怎么顺势"。

### 2. 势—道—术 速览

用 3 张图（趋势雷达 + 时间线 + 机会矩阵）浓缩全报告。

### 3. 势：国家在做什么

- 当前所处周期 + 核心战略方向
- 关键文件/会议（5-8 条带链接与日期）
- 与上轮规划对比的连续性与变化

### 4. 道：产业链如何重塑

- 鼓励类行业：钱、政策、人、技术如何聚集
- 限制/淘汰类行业：时间表、过渡期、转型方向
- 中性/灰度类行业：边界与机会
- 产业链关键节点（Mermaid 图）

### 5. 术：六大场景的具体动作

按用户选定的场景展开，每个场景包含：
- **机会判断**：该场景下国家趋势给的窗口
- **可执行动作**：1-3 个月内可启动的具体动作
- **关键指标**：可观察的领先指标（数据/政策/招聘/订单）
- **风险与避坑**：3 类风险
- **自查清单**：5 条 yes/no 自查

### 6. 历史对照与理论依据

- 1-3 个历史案例（上次规划、相似国家、相似行业）
- 理论框架：参考 [theoretical-foundations.md](references/theoretical-foundations.md)

### 7. 风险与免责声明

> 政策风向可能转变、个人决策应自负风险。本报告仅供参考，不构成投资/职业/学习建议。

## Theoretical Foundations（理论依据）

报告必须基于以下理论的至少 3 个，不可凭空推演。详见 [theoretical-foundations.md](references/theoretical-foundations.md)：

- **比较优势理论**（Ricardo）：顺势 = 把国家比较优势当作个人机会的放大器
- **产业生命周期**（Porter）：导入/成长/成熟/退坡 4 阶段判断入场时机
- **创新扩散理论**（Rogers）：早期采纳者吃红利，晚期买单
- **政府与市场关系**（林毅夫新结构经济学）：有效市场 + 有为政府
- **双循环与逆全球化**（当前语境）：内循环为主、外循环为辅
- **卡尼曼双系统**：避免被单边叙事（"国家让干啥就干啥"）带偏

## Quality Standards（质量铁律）

### 必须做

- ✅ 用 `mcp__MiniMax__web_search` 抓数据，**不靠训练记忆**（避免瞎编）
- ✅ 至少 3 个事实带数字/出处/时间

- ✅ 每个机会标 3 类风险

- ✅ SVG 全部外挂引用
- ✅ markdown 写入 `markdown/` 目录

- ❌ 用内置 `web_search`（违反 CLAUDE.md 全局规则）
- ❌ 写"国家让干啥就干啥"式的单边推荐

- ❌ 凭印象写"普遍认为..."、"业内人士..."等无源叙述
- ❌ 凑数据/凑图表
- ❌ 把训练记忆里的旧政策/旧数据当现状用
- ❌ 忽略用户画像直接给"通用建议"

## Fallback Rules

| 症状 | 一线修复 | 兜底 |
|---|---|---|
| `mcp__MiniMax__web_search` 不可用 | 告知用户并停止 | 不允许使用内置 `web_search` 替代 |
| 用户不给画像 | 按"普通人"画像推演 | 在报告中显式标注"按普通人画像推演" |
| 政策/数据搜不到 | 标【待核】+ 时间戳 | 整条降级为"待核"，不放在主推论里 |
| markdown 目录不存在 | 自动创建 | 告知用户并继续 |
| SVG 渲染失败（浏览器/编辑器不支持） | 同步提供 Mermaid / ASCII 版本 | 在图下方加文字版降级 |
| 抓取结果都是旧政策（> 1 年） | 调整关键词（加"2026"、"十五五"） | 标【时效性不足】+ 显式提示用户 |
| 用户中途改场景 | 重新跑 Step 4-6 | 🛑 STOP：不交付，回到 Checkpoint 0 |
| 6 场景都搜不到机会 | 检查用户画像 vs 国家重点 | 给"等待期"建议 + 提供 1-2 个间接相关机会 |

## ❌ 反例黑名单

| 反模式 | 替代做法 |
|---|---|
| "国家让干啥就干啥" | 强调顺势 + 风险对冲 + 周期判断 |
| 把"卡脖子"等同于"全面国产替代" | 区分关键节点 vs 完整产业链 |
| 抄五年规划原文 | 解读 + 影响 + 案例 |
| 推荐具体个股/个基代码 | 推荐行业/板块/方向（合规边界） |
| 一张图重复出现 | 每张图服务一个独立推论点 |
| 用内联 `<svg>` | 外挂 .svg 文件 + 引用 |
| "未来 5 年一定涨" | "在该政策周期内可能受益" + 退坡期标注 |
| 把规划当股票推荐 | 把规划当结构变化的指示器 |
