# 政策信号分类手册（Policy Signal Taxonomy）

> 把国家政策与新闻拆解成 7 大类信号。每条报告必须按此分类抽取事实。

## 7 大信号分类

### 1. 财政信号（钱从哪来 / 到哪去）

**核心问题**：财政赤字、专项债、转移支付、产业基金的钱从哪来、流向哪？

**抓取关键词**：
- "中央财政赤字"、"地方专项债"、"超长期特别国债"
- "产业引导基金"、"母基金"、"创投基金"
- "减税降费"、"留抵退税"、"研发费用加计扣除"
- "转移支付"、"财政补贴"、"以旧换新补贴"

**判读方法**：
- 赤字率上升 = 国家加杠杆，宽松信号
- 专项债额度向 X 行业倾斜 = X 行业有政策红利
- 产业基金规模超 Y 亿 = 国家级赛道确认
- 补贴退坡 = 行业进入成熟/退坡期

**示例**：
- 2024 年 1 万亿超长期特别国债（"两重"建设）→ 基建 + 科技
- 新能源车购置税减免延续到 2027 年 → 政策周期延长

### 2. 货币信号（钱多钱少 / 利率汇率）

**核心问题**：货币政策是松是紧？人民币是升是贬？流动性往哪走？

**抓取关键词**：
- "LPR"、"MLF"、"存款准备金率"、"逆回购"
- "降息"、"降准"、"加息"、"加息"
- "人民币汇率"、"外汇储备"、"中美利差"
- "社融"、"M2"、"信贷投放"

**判读方法**：
- LPR 下调 = 宽松信号，利好地产/基建/股市
- 降准 = 释放长期资金，利好银行/地产
- 人民币贬值压力 = 利好出口，不利进口
- 社融超预期 = 经济活力强，市场风险偏好上升

**注意**：货币政策 ≠ 财政政策，要分开读。

### 3. 产业信号（鼓励 / 限制 / 淘汰 / 试点）

**核心问题**：国家在鼓励什么、限制什么、淘汰什么？

**抓取关键词**：
- "产业政策"、"行动计划"、"白皮书"
- "鼓励类"、"限制类"、"淘汰类"
- "试点城市"、"示范项目"、"标杆案例"
- "产能压减"、"能耗双控"、"碳排放"

**判读方法**：
- 鼓励类 → 政策红利期，可加码
- 限制类 → 谨慎入场，关注存量改造
- 淘汰类 → 退出或彻底转型
- 试点类 → 关注扩散可能性

**关键识别**：
- 文件中出现"鼓励"、"支持"、"加快发展" → 鼓励
- 出现"有序退出"、"严控"、"压减" → 限制/淘汰
- 出现"试点"、"示范"、"先行先试" → 潜在鼓励（但有不确定性）

### 4. 区域信号（哪个城市/地区被点名）

**核心问题**：政策红利落到哪个城市群、省份、都市圈？

**抓取关键词**：
- "都市圈"、"城市群"、"湾区"、"新区"
- "先行示范区"、"自贸港"、"自贸试验区"
- "对口支援"、"东西部协作"
- "国家中心城市"、"副省级城市"

**判读方法**：
- 城市被点名 = 资源/政策/项目聚集
- 都市圈规划 = 房地产/基建/就业机会
- 自贸港/区 = 贸易/金融/物流机会

**示例**：
- 雄安新区 = 北京非首都功能疏解，长周期
- 粤港澳大湾区 = 科技创新 + 制度开放
- 长三角一体化 = 高端制造 + 数字经济

### 5. 科技信号（卡脖子 / 攻关 / 转化）

**核心问题**：国家在攻关什么技术？哪些方向有突破？

**抓取关键词**：
- "卡脖子"、"攻关"、"重大科技专项"
- "基础研究"、"应用研究"、"成果转化"
- "新型举国体制"、"国家实验室"
- "AI"、"大模型"、"半导体"、"生物医药"、"新能源"、"量子"

**判读方法**：
- 列入"卡脖子"清单 = 长期攻关 + 国产替代
- 重大专项启动 = 资金/人才聚集
- 成果转化政策 = 创业/就业机会

**示例**：
- 半导体设备/材料 → 长期国产替代
- 大模型/AI → 政策+资本+人才三重聚集
- 商业航天/低空经济 → 新质生产力代表

### 6. 民生信号（就业 / 教育 / 医疗 / 住房）

**核心问题**：就业、教育、医疗、住房政策怎么走？

**抓取关键词**：
- "稳就业"、"高校毕业生"、"农民工"
- "双减"、"高考改革"、"职业教育"
- "医保"、"集采"、"分级诊疗"
- "房地产"、"保障房"、"租购并举"

**判读方法**：
- 就业优先战略 = 制造业/服务业稳定岗位
- 职教改革 = 技能型人才需求上升
- 集采常态化 = 仿制药利润压缩，创新药/医疗器械受益
- 房地产新模式 = 保障房 + 商品房 + 租赁三轨

### 7. 对外信号（合作 / 脱钩 / 出海）

**核心问题**：对外关系、贸易、出海的方向是什么？

**抓取关键词**：
- "一带一路"、"中欧班列"、"RCEP"
- "出口"、"外贸"、"外资准入"
- "脱钩"、"断供"、"实体清单"
- "出海"、"全球化"、"供应链"

**判读方法**：
- 一带一路项目落地 = 工程/能源/物流机会
- RCEP/CPTPP = 区域贸易机会
- 脱钩压力 = 国产替代加速
- 出海鼓励 = 跨境电商/SaaS/制造业海外建厂

## 抽取流程

每份报告应按此流程抽取信号：

1. **广撒网**：用 `mcp__MiniMax__web_search` 抓 30-50 条原始信息
2. **粗筛**：按"是否影响行业/区域/人群"过滤到 15-20 条
3. **细筛**：按 7 类信号分类，每类保留 3-5 条
4. **验证**：每条标【出处】+【日期】+【数据标签】
5. **汇总**：7 类信号各 1 张表格，组成"信号清单"

## 数据标签

- 【实】 官方数据已发布
- 【预】 政策刚出，影响待观察
- 【估】 行业研报/分析师估算
- 【待】 报道但未交叉验证
