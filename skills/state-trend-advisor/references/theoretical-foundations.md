# 理论依据手册

> 报告推演必须基于以下理论中的至少 3 个，不可凭空推演。

## 1. 比较优势理论（Ricardo, 1817）

**核心**：每个国家/个人应该专注于自己相对别人效率最高的环节。

**应用**：
- **国家层面**：识别国家比较优势产业 = 政策持续支持的方向
- **个人层面**：把国家比较优势当作个人机会的放大器
  - 国家强 AI → 你学 AI = 顺势
  - 国家强制造 → 你做制造 = 顺势
  - 国家弱地产 → 你别押地产 = 顺势

**操作**：
- 查五年规划/产业政策里的"鼓励类" → 国家比较优势
- 对比用户技能/经验 → 个人比较优势
- 找交集 = 顺势机会

## 2. 产业生命周期（Porter, 1980s）

**核心**：每个产业都经历 4 个阶段：导入 → 成长 → 成熟 → 退坡。

**应用**：
- 识别国家当前主导产业处于哪个阶段
- 不同阶段有不同策略

| 阶段 | 特征 | 政策动作 | 个人策略 |
|---|---|---|---|
| **导入期** | R&D 投入大、产出小 | 强力补贴、税收优惠 | 早期布局，吃红利 |
| **成长期** | 增长快、竞争激烈 | 规范+扶持头部 | 跟头部，谨慎选股 |
| **成熟期** | 增速放缓、利润集中 | 维持+防内卷 | 选龙头，看分红 |
| **退坡期** | 补贴退坡、产能过剩 | 引导转型 | 退出/转型/出海 |

**操作**：
- 查行业增速、产能利用率、补贴政策 → 判断阶段
- 不同阶段对应不同动作（导入期做早期投资、退坡期做转型）

## 3. 创新扩散理论（Rogers, 1962）

**核心**：创新扩散分 5 阶段：创新者 → 早期采纳者 → 早期大众 → 晚期大众 → 落后者。

**应用**：
- 早期采纳者（2.5-16%）吃红利
- 晚期大众（84% 之后）接盘

**操作**：
- 查国家政策/产业基金 → 判断"早期/晚期"
- 早进入（早期采纳者）= 政策红利 + 市场空白
- 晚进入（晚期大众）= 政策退坡 + 竞争红海

**示例**：
- 2009-2015 新能源车：早期采纳者吃到补贴 + 估值
- 2018-2020 互联网金融：晚期大众接盘 P2P
- 2020-2023 半导体：早期采纳者吃国产替代红利
- 2024+ AI：大模型赛道已拥挤，需精细化选

## 4. 新结构经济学（林毅夫, 2000s）

**核心**：有效市场 + 有为政府 = 增长引擎。

**应用**：
- 政策方向 = 国家希望发展的比较优势产业
- 政府通过产业政策、基础设施、教育等降低交易成本
- 个人/企业跟随国家比较优势 = 顺势

**操作**：
- 看国家"鼓励类"产业 → 推断国家比较优势
- 看国家"限制类"产业 → 推断国家想升级/转型的方向
- 看国家"试点"产业 → 推断国家想试错的方向

**示例**：
- 中国"鼓励类"：半导体、新能源、AI、生物医药、高端制造
- 中国"限制类"：高耗能、传统地产、低端制造
- 中国"试点"：数据要素、量子计算、合成生物

## 5. 双循环与逆全球化（当前语境）

**核心**：内循环为主（国内大市场）+ 外循环为辅（高水平开放）。

**应用**：
- 内循环：消费、基建、国产替代
- 外循环：高质量出海（不是低端出口）、制度型开放

**操作**：
- 内循环机会：消费升级、银发经济、医疗、住房新模式
- 外循环机会：高端制造出海、跨境电商、新能源车出海
- 脱钩风险行业：依赖美国技术/市场的行业

**示例**：
- 跨境电商：SHEIN/Temu 模式，吃全球低成本供应链红利
- 新能源车出海：比亚迪/蔚来在欧洲/东南亚建厂
- 半导体：国产替代 + 自主可控

## 6. 卡尼曼双系统（Kahneman, 2011）

**核心**：人脑有 2 套系统：
- 系统 1：快速、直觉、易被带偏
- 系统 2：慢速、理性、需刻意

**应用**：
- 避免被"国家让干啥就干啥"带偏（系统 1）
- 强制用系统 2 做风险评估

**操作**：
- 看到"国家主推 X" → 不要立即冲
- 用 5 个问题自检：
  1. 政策周期到哪了？（导入/成长/成熟/退坡）
  2. 巨头已经入场了吗？
  3. 自身能力匹配吗？
  4. 退坡时怎么退出？
  5. 是否有更优替代？

**示例**：
- 双减前教培行业：政策一来立即退坡
- 2015 O2O：政策鼓励 → 一拥而上 → 大批倒闭
- 2021-2022 互联网平台反垄断：政策转向 → 估值暴跌

## 7. 5 Whys 分析法（大野耐一, 丰田）

**核心**：连续问 5 次"为什么"，找到根本原因。

**应用**：
- 用户问"国家主推 X，我该不该做？"
- 不直接答"做/不做"
- 问 5 次为什么，找到真实需求

**操作**：
```
Q: 国家主推 AI，要不要转行 AI？
Q1: 为什么是 AI？→ 因为有政策红利
Q2: 为什么有政策红利？→ 因为国家想突破卡脖子
Q3: 为什么想突破卡脖子？→ 因为外部技术封锁
Q4: 为什么技术封锁？→ 因为地缘政治竞争
Q5: 为什么地缘政治？→ 因为国家发展到了某个阶段
→ 根本：长期国运方向是科技自立，短期波动会有
→ 推论：AI 是 10 年方向，但短期会有退坡和洗牌
```

## 8. 反事实思考（Counterfactual）

**核心**：思考"如果不做这件事，会发生什么？"

**应用**：
- 评估机会时不要只看"做能赚多少"
- 还要看"不做会失去什么"

**操作**：
- 做 X 机会的预期收益 = E1
- 不做 X 的机会成本 = E0
- 决策 = max(E1, E0, E_other)

**示例**：
- 选择 A 行业，预期年化 10%
- 不选 A，继续 B 行业，预期年化 8%
- 决策：A（如果风险匹配）

## 9. 风险调整后收益（Sharpe Ratio）

**核心**：高收益 ≠ 好机会，要看风险调整后收益。

**应用**：
- 不能只比预期收益
- 还要比收益/风险的比值

**操作**：
- 收益 - 无风险利率 = 超额收益
- 超额收益 / 波动率 = Sharpe Ratio
- 选 Sharpe Ratio 高的，不选预期收益最高的

**示例**：
- A 行业：预期 20%，波动 30% → Sharpe = (20-3)/30 = 0.57
- B 行业：预期 12%，波动 10% → Sharpe = (12-3)/10 = 0.9
- 决策：B 更优（风险调整后）

## 10. 反身性（Soros, 1987）

**核心**：市场预期会反过来影响基本面，形成正反馈或负反馈。

**应用**：
- 政策预期 → 资金流向 → 行业基本面 → 政策兑现
- 注意：预期和基本面的"自我强化"和"自我证伪"

**操作**：
- 当所有人都看好 → 警惕
- 当所有人都看空 → 机会
- 但要识别"非理性繁荣"和"非理性恐慌"

**示例**：
- 2020 新能源：预期一致 → 一致看好 → 2021 短暂泡沫 → 2022 调整
- 2024 AI：预期分歧 → 部分看好 → 持续上行

## 理论使用规范

每份报告：
- ✅ 至少引用 3 个理论
- ✅ 标注理论名称 + 在哪个推论点用
- ❌ 不允许"凭空推演"（没有理论支撑的推论）
- ❌ 不允许"普遍认为"（用理论代替情绪化判断）

## 推论点 → 理论 速查

| 推论点 | 适合理论 |
|---|---|
| 行业进入时机 | 产业生命周期、创新扩散 |
| 行业空间 | 比较优势、新结构经济学 |
| 个人机会 | 比较优势、风险调整后收益 |
| 风险识别 | 卡尼曼双系统、反身性 |
| 退出时机 | 反身性、创新扩散 |
| 长期方向 | 双循环、新结构经济学 |
| 短期波动 | 反事实思考、5 Whys |
