# 用户画像 → 机会映射矩阵

> 把用户输入的"擅长、特长、爱好、资源、城市"翻译成"与国家趋势的匹配度"。

## 画像字段

| 字段 | 含义 | 示例 | 权重 |
|---|---|---|---|
| 擅长领域 | 行业经验/专业背景 | AI、电商、医疗 | 高 |
| 特长/技能 | 可迁移的硬技能 | 算法、销售、外语 | 高 |
| 爱好 | 长期投入意愿方向 | 二次元、户外、阅读 | 中 |
| 现有资源 | 资金/渠道/人脉 | 50万启动金、行业人脉 | 高 |
| 城市/地域 | 行业聚集与政策试点 | 北京、深圳、苏州 | 中 |
| 风险偏好 | 稳健/平衡/激进 | 平衡 | 中 |
| 时间窗 | 1 年/3 年/5 年/10 年 | 3 年 | 高 |

## 默认画像（用户未输入时）

> **25-35 岁、本科以上、一二线城市、可支配 10-100 万、无强行业积累、风险偏好平衡、3 年时间窗**

处理流程：
1. 在报告中显式标注："按普通人画像推演"
2. 给"通用品类"建议（消费、互联网、基础服务、基础技能）
3. 不假设特定行业经验
4. 重点给"低门槛高顺周期"机会

## 匹配度评分

每个机会按以下公式打分（满分 100）：

```
匹配度 = 行业经验分(30) + 技能匹配分(25) + 城市匹配分(15) 
       + 资源匹配分(15) + 风险匹配分(10) + 时间窗匹配分(5)
```

| 分数 | 等级 | 建议 |
|---|---|---|
| 80-100 | 强匹配 | 重点投入 |
| 60-79 | 中等匹配 | 谨慎试点 |
| 40-59 | 弱匹配 | 储备观望 |
| < 40 | 不匹配 | 不建议 |

## 机会 → 行动 翻译

把"匹配机会"翻译成"具体行动"：

| 匹配度 | 行动建议 |
|---|---|
| 80-100 | 立即开始，1 个月内启动 |
| 60-79 | 3 个月内启动，先做 MVP/试点 |
| 40-59 | 6 个月观察期，每月评估 |
| < 40 | 不行动，或转为学习/储备 |
