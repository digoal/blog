# 五维分析框架(详细规则)

> 这是 SKILL.md 步骤 3 的展开。每个维度都有"打分细则、典型证据、常见陷阱"三部分。
> 写预测前必须先 Read 本文件,确保每条打分都有据可依。

## 总览

| 维度 | 权重 | 核心问题 | 打分范围 |
|---|---|---|---|
| 基本面催化 | 30% | 有没有"非交易型"的市场叙事变化? | -2 ~ +2 |
| 技术面结构 | 25% | 价格在哪个关键位?趋势是否健康? | -2 ~ +2 |
| 资金面 | 20% | 大钱在流入还是流出? | -2 ~ +2 |
| 情绪面 | 15% | 市场是贪婪还是恐慌?是不是一致预期? | -2 ~ +2 |
| 宏观面 | 10% | 大盘环境是否配合个股走势? | -2 ~ +2 |

加权求和后,最终分映射到方向与概率(见 SKILL.md 步骤 4)。

## 维度 1:基本面催化(权重 30%)

### 打分细则

| 打分 | 触发条件 |
|---|---|
| **+2** | 财报大幅超预期(EPS beats by >10%)+ 指引上修,或重磅催化剂(并购、回购 ≥ 5% 市值、FDA 批准) |
| **+1** | 业绩小幅超预期 / 利好消息已被部分 price-in / 行业级正面催化 |
| **0** | 无重大消息 / 财报发布超过 5 个交易日 / 消息已被市场消化 |
| **-1** | 业绩小幅不及预期 / 行业负面消息 / 高管离职 |
| **-2** | 业绩暴雷(EPS miss by >10%)+ 指引下调,或重磅利空(产品召回、重大诉讼、SEC 调查) |

### 典型证据来源

- SEC EDGAR 8-K、10-Q
- 公司投资者关系页(earnings release、transcript)
- Bloomberg / Reuters / CNBC 财报快讯
- Wallstreetcn 美股、华尔街见闻
- Seeking Alpha earnings preview

### 常见陷阱

1. **"Beats" 不等于强利多** — 实际 EPS $1.50 vs 预期 $1.49,只高 1 美分,基本无效
2. **Revenue Beats vs EPS Beats 关注度不同** — 市场对 EPS 更敏感(直接算 PE)
3. **Guidance 才是关键** — 即便 EPS beats,但指引下调,股价照样跌
4. **回购 < 1% 市值 = 装饰性回购**,对股价影响有限
5. **公司行动公告后 24h 内** — 价格已部分反应,需评估"price-in 程度"

### 通俗话术转换(用于写报告)

- "EPS beats consensus" → "每股盈利(EPS)比分析师平均预期高"
- "guidance raised" → "公司上调了对未来几个季度的业绩指引,意味着管理层自己都觉得生意会比之前预期的好"
- "buyback authorization" → "公司董事会批准了股票回购计划,会用自己的现金在市场上买回自己的股票"

## 维度 2:技术面结构(权重 25%)

### 打分细则

| 打分 | 触发条件 |
|---|---|
| **+2** | 突破关键阻力(如 200 日均线 / 多年盘整上沿) + 成交量放大 1.5x 以上 |
| **+1** | 上升通道内,价格在 20 EMA 之上,趋势健康 |
| **0** | 区间震荡,无明确方向 |
| **-1** | 下降通道内,价格在 20 EMA 之下 |
| **-2** | 跌破关键支撑(如前低 / 200 日均线 / 盘整下沿) + 放量 |

### 必须看的 4 个层

1. **大周期(月/周)**:200 日均线方向,过去 3-6 个月趋势
2. **中周期(日线)**:近 20 日形态、前高前低、跳空缺口
3. **小周期(60min/15min)**:盘前交易区间、关键支撑/压力
4. **成交量**:5 日均量 vs 60 日均量,放量/缩量信号

### 核心工具与判读

- **VWAP**(成交量加权均价):价格在 VWAP 上 = 当日多头主导,下 = 空头主导
- **关键位**:前一日高低点、前日 VWAP、近 20 日 EMA、布林带上下轨
- **形态**:突破回踩、上升三角形、双底、双顶、头肩顶
- **RS(相对强度)**:个股 vs SPY 的相对强度

### 常见陷阱

1. **不要用 RSI 单一指标做反向判断** — 强势股 RSI 长期 > 80,弱势股长期 < 20
2. **突破假突破占 50%** — 必须等"突破 + 回踩不破"才确认
3. **支撑/压力不是精确数** — 通常是"区间"而非"点位"
4. **缩量整理 ≠ 看空** — 缩量盘整是健康的中继形态
5. **跳空缺口的回补规律** — 普通缺口 70% 会在 3-5 日内回补,突破/逃逸缺口例外

### 通俗话术转换

- "200-day moving average" → "200 日均线(过去 200 个交易日的平均收盘价,被很多机构视为长期牛熊分界线)"
- "VWAP" → "成交量加权均价(把当天的成交按价格加权算出来的'平均成本',机构常用它看当天谁占主导)"
- "breakout" → "突破(股价涨过之前一直没能涨过的关键价位)"
- "RSI" → "相对强弱指数(0-100,衡量股价涨/跌的力度,常被用来判断超买超卖)"

## 维度 3:资金面(权重 20%)

### 打分细则

| 打分 | 触发条件 |
|---|---|
| **+2** | 巨额回购(>2% 市值)+ 内部人增持 + OI 看涨异动 + 暗池净流入 |
| **+1** | 内部人增持 / OI 看涨 / 机构 13F 加仓 |
| **0** | 无明显异动 |
| **-1** | 暗池出货 / 内部人减持 / OI 看跌异动 |
| **-2** | 高管集体减持 + 暗池异常 + 大股东清仓式减持 |

### 三类资金信号

#### 3.1 机构持仓(13F)
- 滞后 45 天,只能识别主流共识
- 顶级对冲基金(Bridgewater、Citadel、Renaissance)同向加仓 = 强信号

#### 3.2 盘后大宗交易与暗池
- 盘后大宗溢价/折价 > 5% = 利益相关方在"加急出货/进货"
- 暗池(ATS)成交量 > 30% = 警惕

#### 3.3 期权市场(资金面最重要信号)
- **OI(未平仓合约)变化**:某执行价 OI 暴增 = "庄家级"资金建仓
- **大单扫描**:>50 万美元、远月(LEAPS)期权,通常是机构行为
- **Put/Call 比率**:
  - 总 Put/Call > 1.2 = 极度恐慌(可能反而是底部)
  - 总 Put/Call < 0.7 = 极度贪婪(风险升高)
  - 个股 Put/Call < 0.3 = 多头一致看涨
- **IV Rank**:
  - > 50 = 高隐含波动率,适合卖期权
  - < 30 = 低隐含波动率,适合买期权
- **Max Pain**:期权到期日时做市商希望收盘价靠近的价位

### 常见陷阱

1. **13F 滞后** — 千万不要用 13F 做"实时决策"
2. **散户大单 ≠ 机构大单** — 期权扫描要区分"lots 数量"和"notional 美元金额"
3. **IV 跳升未必是方向信号** — 可能是财报前对冲(gamma hedging)
4. **OI 暴增要分清 Call/Put** — 看涨 OI 暴增是利好,看跌 OI 暴增是利空
5. **内部人 Form 4** 也要看金额,$10K 增持和 $10M 增持意义完全不同

### 通俗话术转换

- "13F filing" → "持仓披露(美国证监会要求管理资产 1 亿美金以上的机构每季度披露一次自己买了什么)"
- "dark pool" → "暗池(一种不公开显示订单的另类交易系统,大机构常用它悄悄买卖)"
- "put/call ratio" → "看跌/看涨期权比(市场里看跌合约数量除以看涨合约数量,常用来衡量市场恐慌程度)"
- "open interest" → "未平仓合约(市场上还没平掉的期权合约总数,反映资金对该方向的押注)"
- "IV crush" → "隐含波动率骤降(财报等事件过后,期权价格里'不确定性'的成分突然消失,导致期权买方大亏)"

## 维度 4:情绪面(权重 15%)

### 打分细则

| 打分 | 触发条件 |
|---|---|
| **+2** | 分析师一致上调目标价 + 内部人增持 + 散户关注度低(未被过热炒作) |
| **+1** | 分析师上调 + 散户关注度适中 |
| **0** | 情绪中性 / 极度一致预期(无论多空,一致预期都是反向风险) |
| **-1** | 分析师下调 / 散户关注度激增(反向风险) |
| **-2** | 业绩暴雷 + 分析师集体下调 + 散户踩踏 |

### 关键指标

#### 4.1 散户情绪(反向指标为主)
- Reddit WSB、X(Twitter) Stocktwits 关注度激增 = 散户涌入,常是反向
- AAII 投资者情绪调查:看多 > 50% = 警惕,< 20% = 可能见底
- CNN 恐惧贪婪指数:极端贪婪 > 80 反向看空,极端恐惧 < 20 反向看多

#### 4.2 卖方分析师态度
- 评级调整:Upgrades > Downgrades 连续 5 天 = 多头氛围
- 目标价调整:多家投行同时上调目标价 = 强信号
- Pre-announcement(业绩预告):尤其负向预告,股价已跌 10%,次日可能再跌 3-5%

#### 4.3 内部人交易(优先级最高)
- CEO/CFO 增持(>$100K) = 极强信号
- 大量内部人减持 + 节奏加快 = 预警

### 常见陷阱

1. **散户关注度激增 = 反向** — 尤其中小盘 meme 股
2. **AAII 极端值有用** — 但中性区间(20%-50%)预测力差
3. **一致预期是反向风险** — 所有人都看多时,反而要警惕
4. **Form 144 ≠ 一定负面** — 可能只是个人理财

### 通俗话术转换

- "consensus" → "一致预期(分析师对公司业绩的预测平均数,常被用来衡量实际业绩的'惊喜程度')"
- "Form 4" → "内部人交易披露(公司高管、董事买卖自家股票时,2 个工作日内要向 SEC 报备)"
- "pre-announcement" → "业绩预告(公司在正式财报前先发出大致数字,通常是因为和预期差距太大必须提前说)"

## 维度 5:宏观面(权重 10%)

### 打分细则

| 打分 | 触发条件 |
|---|---|
| **+2** | 期货涨 > 1% + VIX 跌 > 10% + 风险偏好全面回升 |
| **+1** | 期货小涨 + VIX 小跌 + 10Y 收益率稳定 |
| **0** | 中性 / 大事件日(CPI/FOMC/非农)系统风险主导 |
| **-1** | 期货小跌 + VIX 小涨 + 收益率上升 |
| **-2** | 期货跌 > 1% + VIX 飙升 > 20% + 系统性恐慌 |

### 关键变量

- **ES 期货 / NQ 期货**:盘前涨跌幅直接预测大盘开盘方向
- **VIX**:> 30 = 恐慌,15-25 = 正常,< 15 = 风险偏好高
- **10Y 美债收益率**:单日 +10bp 通常砸科技股 2-3%
- **DXY 美元指数**:美元强 = 跨国企业利润受损,弱 = 利好
- **次日宏观日历**:CPI/PPI/PCE/FOMC/非农公布日 = 波动率天然放大

### 常见陷阱

1. **VIX 不能预测方向** — VIX 衡量的是"波动幅度"而非"涨跌方向"
2. **大事件日不要重仓** — 系统性风险主导个股
3. **期货夜间变化 ≠ 次日收盘** — 跳空后常被"mean reversion"拉回
4. **"risk-on" / "risk-off"** — 当资金从科技/小盘流入防御(公用事业、必需消费),是 risk-off 信号

### 通俗话术转换

- "VIX" → "恐慌指数(衡量市场对未来 30 天波动幅度的预期,数字越大说明大家越害怕)"
- "10-year yield" → "10 年期美债收益率(美国政府借 10 年期钱的利率,通常被视为'无风险利率',对全市场估值有锚定作用)"
- "DXY" → "美元指数(美元对欧元、日元等一揽子货币的强弱,常影响跨国公司的报表利润)"
- "CPI" → "消费者物价指数(衡量老百姓买买买的东西变贵了多少,是美联储决定加不加息的核心数据)"
- "FOMC" → "美联储公开市场委员会(决定美国基准利率的会议,每次会议结果都会引发市场大幅波动)"

## 加权求和 → 方向与概率

```
总分 = Σ(每维打分 × 权重)
     = 基本面 × 0.30 + 技术面 × 0.25 + 资金面 × 0.20 + 情绪面 × 0.15 + 宏观面 × 0.10
```

| 总分区间 | 方向 | 置信度 | 基础概率调整 |
|---|---|---|---|
| > +1.0 | 强看多 | 高 | +10% ~ +15% |
| +0.3 ~ +1.0 | 弱看多 | 中 | +3% ~ +8% |
| -0.3 ~ +0.3 | 中性震荡 | 中低 | 0% |
| -1.0 ~ -0.3 | 弱看空 | 中 | -3% ~ -8% |
| < -1.0 | 强看空 | 高 | -10% ~ -15% |

**最终方向概率** = 55%(基础)+ 调整,clamp 到 [35%, 65%]

**强制规则**:
- 大事件日(CPI/FOMC)基础概率 clamp 到 [45%, 55%](信息已被充分 price-in)
- 财报次日(自家财报)概率区间扩到 [30%, 70%](波动率天然大)
- 信息严重缺失时,clamp 到 [45%, 55%](不基于缺失数据强行判断)
