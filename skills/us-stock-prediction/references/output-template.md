**备选方案:**

### 5.2 止损策略

- **止损价**: ${X}
- **止损理由**: {跌破 ____ 支撑 / 形态破坏 / 关键事件反向}
- **最大可承受亏损**: 账户净值的 {X}%(确保此笔交易最大亏 = 账户 1%)

**仓位计算:**
```
单笔仓位(股数) = (账户净值 × 风险预算%) ÷ (入场价 − 止损价)
             = ($100,000 × 1%) ÷ (${入场} − ${止损})
             = {X} 股
             = 总仓位 ${X}(账户净值的 {X}%)
```
