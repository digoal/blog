# 术语通俗解释词典

> 写预测报告时,每出现一个专业术语,都必须用下面的话术"翻译"成小白能懂的语言。
> 第一次出现时用括号补充,后续可省略。
> 目标读者:投资小白(知道 K 线、PE,但不知道 OI、IV、暗池、IV crush 是什么)。

## 财报与基本面类

| 术语 | 通俗解释 |
|---|---|
| EPS(Earnings Per Share) | 每股收益 = 公司净利润 ÷ 总股本,衡量"每一股赚了多少钱" |
| Revenue / Sales | 营业收入,公司卖东西赚的总收入(还没扣成本) |
| Net Income | 净利润,扣完所有成本、税费之后真正到手的钱 |
| Guidance | 公司对下个季度/年度的业绩展望,管理层自己说"我们预计..." |
| Beats / Misses Consensus | 实际业绩比分析师平均预期"高/低" |
| Beat by $X | 比预期高 $X(可以是 EPS、Revenue 等) |
| Forward P/E | 远期市盈率,用"未来 12 个月的预期 EPS"算的 PE |
| Trailing P/E | 静态市盈率,用"过去 12 个月的真实 EPS"算的 PE |
| 8-K Filing | 重大事件披露(美国证监会要求公司发生重大事件时 4 天内公开披露) |
| 10-Q / 10-K | 季度报告 / 年度报告 |
| Buyback / Repurchase | 股票回购(公司用自己的现金在市场上买回自己的股票,通常会推高股价) |
| Dividend | 分红,公司把利润的一部分直接发给股东 |
| Stock Split | 拆股(把 1 股拆成多股,Apple 1 拆 4 就是典型例子) |
| Reverse Split | 反向拆股(把多股合并成 1 股,通常是低股价公司避免被退市) |
| M&A / Acquisition | 并购,一家公司买下另一家 |
| Spin-off | 分拆,母公司把子公司独立出去,股东各拿到一份新公司股票 |
| SEC | 美国证券交易委员会(美国的"证监会",负责监管证券市场) |

## 价格与盘面类

| 术语 | 通俗解释 |
|---|---|
| Bid / Ask | 买一价 / 卖一价,你想卖能立刻卖出的最高价 / 你想买能立刻买入的最低价 |
| Spread | 买卖价差,卖一减买一,差越小说明流动性越好 |
| VWAP | 成交量加权均价,把当天的成交按价格加权算出来的"平均成本",机构常用它判断"当天谁占主导" |
| Pre-market / After-hours | 盘前交易 / 盘后交易,正常交易时段(美东 9:30-16:00)之外的交易,流动性差、波动大 |
| Gap Up / Gap Down | 跳空高开 / 跳空低开,今天的开盘价比昨天的收盘价高/低了一大截 |
| Support / Resistance | 支撑位 / 压力位,股价"跌到这容易止跌"的价位 / "涨到这容易回落"的价位 |
| Breakout | 突破,股价涨过之前一直没能涨过的关键价位 |
| Breakdown | 跌破,股价跌穿之前一直撑住的关键价位 |
| Pullback | 回调,股价上涨过程中短暂下跌 |
| Retest | 回测,股价突破关键位后,再次回到那个位置"试探"是否撑得住 |
| Moving Average(MA) | 移动平均线,过去 N 天的平均收盘价,常看的有 20 日、50 日、200 日 |
| EMA | 指数移动平均线,比普通 MA 更重视"近期的价格" |
| RSI(Relative Strength Index) | 相对强弱指数,0-100 的数,衡量股价涨/跌的力度,> 70 通常算超买,< 30 算超卖 |
| MACD | 平滑异同移动平均线,看"短期均线和长期均线的距离变化"判断趋势 |
| Bollinger Bands | 布林带,股价上下两条线,中间一条均线,常用来判断股价是否"超出正常范围" |
| ATR(Average True Range) | 平均真实波幅,衡量股价近期的"平均波动幅度",常被用来设止损 |
| Beta | 贝塔系数,衡量个股相对大盘的波动倍数,Beta=1.5 表示大盘涨 1%,个股平均涨 1.5% |

## 期权类(资金面的核心)

| 术语 | 通俗解释 |
|---|---|
| Call Option | 看涨期权,给持有者"在未来某个时间以某个价格买入"的权利(不是义务) |
| Put Option | 看跌期权,给持有者"在未来某个时间以某个价格卖出"的权利(不是义务) |
| Strike Price | 行权价,期权约定的"未来买卖的价格" |
| Premium | 权利金,买期权要付的"保险费",就是期权的现价 |
| Open Interest(OI) | 未平仓合约,市场上还没平掉的期权合约总数,反映"押注这个方向的总资金" |
| Volume | 成交量,今天新开 + 平仓的总合约数 |
| Put/Call Ratio | 看跌/看涨比,市场里看跌合约数量除以看涨合约数量,常用来衡量市场恐慌程度 |
| IV(Implied Volatility) | 隐含波动率,从期权价格反推出来的"市场对未来波动的预期",数字越大说明大家觉得越要剧烈波动 |
| IV Rank / IV Percentile | 把当前 IV 放在过去 1 年的范围里看排第几,常用来判断"现在期权贵不贵" |
| IV Crush | 隐含波动率骤降,财报等事件过后,期权里"不确定性"的成分突然消失,导致期权买方大亏 |
| LEAPS | 长期期权(到期日在 1 年以上),机构常用它做长期布局 |
| Max Pain | 最大痛点,期权到期日时,做市商"最希望"股价收盘在的位置(让最多期权变废纸) |
| Gamma | 期权的希腊字母之一,衡量"Delta 变化的速度",机构对冲时很关心 |
| Delta | 期权的希腊字母之一,衡量"股价每涨 1 美元,期权涨多少" |
| Hedge | 对冲,用一笔交易来"抵消"另一笔交易的风险 |
| Covered Call | 备兑看涨,持有股票的同时卖出看涨期权(等于"愿意在某个价格卖出") |

## 资金与机构类

| 术语 | 通俗解释 |
|---|---|
| 13F Filing | 持仓披露,美国证监会要求管理资产 1 亿美金以上的机构每季度披露一次自己买了什么 |
| Hedge Fund | 对冲基金,用各种复杂策略做投资的大型机构 |
| Whale | 巨鲸,持仓或交易金额特别大的投资者(机构和个人都算) |
| Dark Pool | 暗池,不公开显示订单的另类交易系统,大机构用它悄悄买卖,普通投资者看不到 |
| Block Trade | 大宗交易,一次性交易大量股票(通常 > 10 万股或 > 20 万美元) |
| Form 4 | 内部人交易披露,公司高管、董事买卖自家股票时,2 个工作日内要向 SEC 报备 |
| Form 144 | 限售股抛售预告,公司内部人计划卖出受限股票时的"提前通知" |
| Insider Buying/Selling | 内部人买入/卖出(自己公司股票) |
| Institutional Ownership | 机构持股比例,机构持有的股份占公司总股本的百分比 |
| Short Interest | 空头持仓,做空该股的资金规模,占流通股本的比例 = 做空比率 |
| Short Squeeze | 轧空,做空的人被迫买回股票止损,推高股价形成"踩踏式上涨" |
| FTD(Failure to Deliver) | 未交割,卖方没能在结算日交出股票(常被怀疑是裸卖空) |

## 宏观与市场环境类

| 术语 | 通俗解释 |
|---|---|
| Fed / Federal Reserve | 美联储,美国的中央银行,决定基准利率 |
| FOMC | 美联储公开市场委员会,决定利率的会议,每次会议结果都引发市场大幅波动 |
| Interest Rate Hike / Cut | 加息 / 降息,美联储提高 / 降低基准利率 |
| CPI | 消费者物价指数,衡量老百姓买买买的东西变贵了多少,是美联储决定加不加息的核心数据 |
| PPI | 生产者物价指数,工厂出厂端的物价,比 CPI 更"上游" |
| PCE | 个人消费支出物价指数,美联储最看重的通胀指标(权重比 CPI 大) |
| NFP / Jobs Report | 非农就业报告,每月第一个周五公布,显示美国新增了多少工作 |
| 10Y / 10-Year Yield | 10 年期美债收益率,美国政府借 10 年期钱的利率,通常被视为"无风险利率",对全市场估值有锚定作用 |
| Yield Curve | 收益率曲线,不同期限美债的利率连成的线,倒挂(短期 > 长期)常被视为衰退信号 |
| DXY | 美元指数,美元对欧元、日元等一揽子货币的强弱,常影响跨国公司的报表利润 |
| VIX | 恐慌指数(正式名"芝加哥期权交易所波动率指数"),衡量市场对未来 30 天波动幅度的预期,数字越大说明大家越害怕 |
| VVIX | VIX 的 VIX,衡量"市场对'波动率本身'的波动预期" |
| S&P 500 / SPX / SPY | 标普 500 指数 / 标普 500 ETF,美国 500 家大公司的加权指数,是大盘的代表 |
| Nasdaq / NDX / QQQ | 纳斯达克指数 / 纳斯达克 100 ETF,科技股为主的指数 |
| Dow / DJIA | 道琼斯指数,30 家大公司(多数是传统行业)的加权指数 |
| Russell 2000 / IWM | 罗素 2000 指数,2000 家小公司,代表小盘股 |
| ES Futures | 标普 500 期货,在 CME(芝加哥商品交易所)交易,流动性最好 |
| NQ Futures | 纳指 100 期货 |
| Risk-on / Risk-off | 风险偏好上升 / 下降,资金从防御板块(公用事业、必需消费)流入风险板块(科技、小盘)叫 risk-on,反之 risk-off |
| Bull / Bear Market | 牛市(指数从低点涨 20%+) / 熊市(指数从高点跌 20%+) |
| Correction | 回调,指数从高点跌 10%-20% |
| Crash | 崩盘,短期(几天到几周)暴跌 20%+ |

## 风险与交易类

| 术语 | 通俗解释 |
|---|---|
| Stop Loss | 止损,股价跌到某个价位就自动卖出的预设指令 |
| Take Profit | 止盈,股价涨到某个价位就自动卖出的预设指令 |
| Trailing Stop | 跟踪止损,止损价随股价上涨自动上移,但不会下移,常用来"锁住利润" |
| Position Sizing | 仓位管理,决定"一笔交易买多少" |
| Risk/Reward Ratio | 赔率(潜在盈利:潜在亏损),2:1 表示赚 2 块最多亏 1 块 |
| Expected Value | 期望值,胜率 × 盈利幅度 − 败率 × 亏损幅度,长期下来能赚多少的"理论值" |
| Drawdown | 回撤,账户从最高点跌下来的幅度,衡量"最惨的时候亏多少" |
| Sharpe Ratio | 夏普比率,(收益 − 无风险利率) ÷ 波动率,衡量"每承担一份风险,多赚多少" |
| Liquidity | 流动性,股票好不好买卖,日均成交量越大、买卖价差越小,流动性越好 |
| Slippage | 滑点,实际成交价比预期差(常见于快速波动 + 流动性差的票) |
| Overweight / Underweight | 超配 / 低配,机构对某只股票"比基准配得更多 / 更少"的评级 |
| Hold / Buy / Sell | 持有 / 买入 / 卖出,分析师的三大基本评级 |

## 使用规则

1. **第一次出现某术语时**,必须用括号补充通俗解释(放在术语后面)
2. **写报告的"六、术语通俗解释"章节** 时,从本文件按出现顺序选 5-10 个最关键的术语列出来
3. **不要试图穷尽所有术语** — 选对小白最重要的、影响判断的关键概念
4. **如果某术语在本文件没有**,先按上面的话术风格自己"翻译",然后在 markdown 中标注"自定义解释",方便后续补充
