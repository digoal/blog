# 张仲景表达 DNA 研究

> 第三篇：仲景文本的"语言指纹"与术语体系
> 视角：写作风格、解构方式、关键术语、句式模板、辨析点
> 用途：为仿写、张仲景视角写作提供表达层参考

---

## 一、研究目标

如果说前两篇（生平考据、对话风格）回答了"仲景是谁、怎么说话"，
那么这一篇要回答的是：**"仲景的文字长什么样"**。

具体而言：
- 他有哪些**标志性句式**？——句子层面的指纹
- 他的**风格维度**如何分布？——文本层面的 DNA
- 他的**术语体系**是什么？——概念层面的指纹
- 后世传抄与**仲景原文**有何出入？——辨析层面的陷阱
- 这些材料都来自**哪些出处**？——可追溯的引用源

本篇是仿写工具的**风格底座**。任何张仲景视角的写作，最终都要回到这一层的句式、词汇、节奏感。

## 二、八大标志性句式

下面是从《伤寒论》《金匮要略》原文中提炼出的八种**反复出现**的句式。
这些不是偶发，而是仲景构建文本的"基本句法单位"。

### 句式 1：「……者，……也」病机判断式

**结构**：`[病象]者，[病机]也`

**原文举例**：
- "**病人脉微而涩者，此为医所病也**。"
- "**病者，身热足寒，颈项强急，恶寒，时头热面赤，目脉赤，独头动摇，卒口噤，背反张者，痉病也**。"
- "**寸口脉浮而紧，紧则为寒，浮则为虚，虚寒相搏，邪在皮肤。浮者血虚，络脉空虚，贼邪不泻，或左或右，邪气反缓，正气即急，正气引邪，㖞僻不遂**。"

**风格特征**：
- 用判断语气"……者，……也"收束，类似古汉语的"判断句"
- 简洁、有结论感，像在写医案中的"病机小结"
- 适合用于：**病机定位、诊断定性、症状归因**

**仿写模板**：
```
[症状群]者，[病机名]也。
[脉象/舌象]者，[病因]也。
[病程描述]者，[预后判断]也。
```

### 句式 2：「……者，……者」分类对举式

**结构**：`[条件A]者，[条件B]者，[分别结论]`

**原文举例**：
- "**太阳病，发热，汗出，恶风，脉缓者，名为中风。太阳病，或已发热，或未发热，必恶寒，体痛，呕逆，脉阴阳俱紧者，名为伤寒**。"
- "**太阳中风，阳浮而阴弱。阳浮者，热自发；阴弱者，汗自出**。"

**风格特征**：
- 用"者"字做对仗、并置、对比
- 风格像"分门别类"——先列条件，再下定义
- 适合用于：**辨证分型、鉴别诊断**

**仿写模板**：
```
[A 证]者，[症状 X]也；
[B 证]者，[症状 Y]也。
[A]者主表，[B]者主里。
```

### 句式 3：「若……者」条件递进式

**结构**：`若[新条件出现]，则[转入下一阶段]`

**原文举例**：
- "**太阳病，若吐、若下、若温针，仍不解者，此为坏病**。"
- "**伤寒五六日，呕而发热者，柴胡汤证具**。"
- "**少阴病，脉微细，但欲寐也**。"

**风格特征**：
- 用"若"引导条件分支
- 节奏短促，类似"如果……那么……"
- 适合用于：**病情演变、阶段推进、转方判断**

**仿写模板**：
```
若[症状加剧]，则[病机转化]；
若[条件成立]，宜[治法 X]。
若不[条件]，则[后果 Y]。
```

### 句式 4：「……宜……汤」方证对应式

**结构**：`[证候群]，宜[方名]`

**原文举例**：
- "**太阳病三日，发汗不解，蒸蒸发热者，属胃也，调胃承气汤主之**。"
- "**伤寒五六日，中风，往来寒热，胸胁苦满，默默不欲饮食，心烦喜呕，或胸中烦而不呕，或渴，或腹中痛，或胁下痞硬，或心下悸，小便不利，或不渴，身有微热，咳者，小柴胡汤主之**。"
- "**发汗后，不可更行桂枝汤，汗出而喘，无大热者，可与麻黄杏仁甘草石膏汤**。"

**风格特征**：
- 极简——一个判断 + 一个方名
- "主之"是**专断语气**（必用此方），"宜"是**商量语气**（推荐此方），"可与"是**许可语气**（可以此方）
- 三种语气有微妙差别：主之 > 宜 > 可与
- 适合用于：**方证收束、治疗决定**

**仿写模板**：
```
[症状群]，[方名]主之。
[病机描述]，宜[方名]。
[证候]，可与[方名]。
```

### 句式 5：「……者，……汤主之」经典方证句

**结构**：`[证候]者，[方名]主之`

**原文举例**：
- "**太阳病，头痛，发热，汗出，恶风者，桂枝汤主之**。"
- "**太阳中风，脉浮紧，发热恶寒，身疼痛，不汗出而烦躁者，大青龙汤主之**。"
- "**少阴病，脉沉者，急温之，宜四逆汤**。"

**风格特征**：
- 这是伤寒论最经典、最高频的句式
- 一句话就是**一个完整的临床决策**
- 短小、有力、有权威感
- 适合用于：**开门见山下处方**

**仿写模板**：
```
[证候 1]、[证候 2]、[证候 3]者，[方名]主之。
```

### 句式 6：「……故也」「是以……」病机推导式

**结构**：`[前置病机]，故[后置结果]`

**原文举例**：
- "**血弱气尽，腠理开，邪气因入，与正气相搏，结于胁下。正邪分争，往来寒热，休作有时，默默不欲饮食。藏府相连，其痛必下，邪高痛下，故使呕也**。"
- "**夫病痼疾，加以卒病，当先治其卒病，后乃治其痼疾也**。"
- "**病者素不应食，而反暴思之，必发热也**。"

**风格特征**：
- 用"故"、"是以"作因果连接
- 展示仲景的**逻辑推导过程**——他不直接说结论，而是先列前因，再下后果
- 适合用于：**推理过程、病机解释、预后说明**

**仿写模板**：
```
[病机描述]，故[症状/结果]。
[条件]，[推导过程]，是以[结论]。
```

### 句式 7：「……者愈」「……者死」「……者难治」预后判断式

**结构**：`[证候/条件]者，[预后]`

**原文举例**：
- "**发汗后，恶寒者，虚故也；不恶寒，但热者，实也。当和胃气，与调胃承气汤**。"
- "**少阴病，下利止而头眩，时时自冒者，死**。"
- "**病六七日，其人发热，暴利，汗出不止者，难治**。"

**风格特征**：
- 直接给出**生死判断**——不绕弯
- "死"、"难治"、"不治"是仲景的"硬话"
- 适合用于：**预后警示、医患沟通、决断时刻**

**仿写模板**：
```
[症状群]者，死。
[病程描述]者，难治。
[病机恶化]者，不治。
```

### 句式 8：「……者，不可……」「……者，勿……」禁忌句式

**结构**：`[证候]者，[禁忌操作]`

**原文举例**：
- "**太阳病，桂枝证，医反下之，利遂不止，脉促者，表未解也；喘而汗出者，葛根黄芩黄连汤主之**。"
- "**呕家有痈脓者，不可治呕，脓尽自愈**。"
- "**疮家，虽身疼痛，不可发汗**。"

**风格特征**：
- 否定句、命令句
- "不可"是**绝对禁止**，"勿"是**提醒**，"反"是**警告误治**
- 适合用于：**治疗禁区、医嘱、误治提醒**

**仿写模板**：
```
[证候]者，不可[治法 X]。
[病机]者，勿[误操作]。
[禁忌条件]者，反[危险操作]则[后果]。
```

## 三、六维风格标签

如果说八大句式是"句子层"的指纹，那这六个维度就是"文本层"的 DNA。
任何一段仲景风格的文字，在这六个维度上都应该有可识别的特征。

### 维度 1：简练（Brevity）

**特征**：
- 一句话常 10-25 字
- 不用形容词，不修饰
- 能用一个字不用两个字

**典型**：
- "脉微细，但欲寐。"（少阴病提纲）—— 8 个字讲清一个证型
- "呕而发热者，小柴胡汤主之。"—— 13 个字，一个处方决定

**对比**：
- 同期医书《黄帝内经》："**阴阳者，天地之道也，万物之纲纪，变化之父母，生杀之本始，神明之府也**。"—— 25 字讲一个抽象概念
- 仲景："**太阳病，脉浮紧，无汗，发热，身疼痛，八九日不解，表证仍在，此当发其汗**。"—— 一个判断 + 一个治法

**仿写提示**：
- 一段话尽量 1-3 句
- 每句控制在 10-25 字
- 不用"非常"、"极其"、"非常地"等修饰

### 维度 2：判断（Judgmental）

**特征**：
- 几乎不使用疑问句
- 大量使用判断句（……者，……也）
- 给出明确结论，不留模糊地带

**典型**：
- "**此为坏病**。"—— 4 字定性
- "**死**。"—— 1 字定生死

**对比**：
- 儒医风格："此病或因……或因……宜审慎辨之"—— 仲景绝不这么写
- 仲景风格："此为坏病，桂枝不中与也。"—— 直接排除

**仿写提示**：
- 表述时用句号，不用问号
- 用"是"、"为"、"主之"等判断词
- 避免"也许"、"或许"、"可能"

### 维度 3：客观（Objectivity）

**特征**：
- 只描述可观察的症状、脉象、舌象
- 不解释"为什么病人会这样"
- 不进入病人心理

**典型**：
- "**病人脉微而涩**"—— 只说脉象
- "**默默不欲饮食**"—— 默默 = 不想说话；不欲饮食 = 不想吃
- "**但欲寐**"—— 只描述"只想睡"

**对比**：
- 现代医学叙事："病人因长期失眠导致情绪低落"—— 进入心理
- 仲景叙事："病人默默不欲饮食"—— 只描述行为

**仿写提示**：
- 只写"看得到、摸得到、问得到"的东西
- 不写"病人觉得……"、"病人担心……"
- 不写"病人似乎……"

### 维度 4：节律（Rhythm）

**特征**：
- 短句为主，长短交错
- 常用四字格、五字格、六字格
- 形成"噼里啪啦"的节拍感

**典型**：
- "**太阳病，头痛，发热，汗出，恶风，桂枝汤主之**。"
- 句式节奏：4 + 2 + 2 + 2 + 2 + 2 + 4 = 四字起步 + 短句连珠 + 四字收束

**对比**：
- 散文风格："当病人出现头痛、发热、汗出、恶风这些症状的时候，可以用桂枝汤来治疗。"
- 仲景风格：4 + 2 + 2 + 2 + 2 + 2 + 4，噼里啪啦一气到底

**仿写提示**：
- 多用顿号（、）连接同类项
- 让句子有"列清单"的节奏
- 避免长定语、长状语

### 维度 5：术语密度（Terminology Density）

**特征**：
- 高频使用固定医学术语
- 一段话常常 5-10 个术语
- 术语之间靠逻辑（不是语法）连接

**典型**：
- "**太阳病，发汗后，脉浮，烦渴者，五苓散主之**。"
- 术语密度：太阳病 / 发汗后 / 脉浮 / 烦渴 / 五苓散 = 5 个术语 / 11 字 ≈ 45% 密度

**对比**：
- 现代医学科普："当你因为感冒发汗以后，如果脉搏比较浅表跳动，并且感到口渴烦躁，这可能是水液代谢出了问题，可以考虑用五苓散来调节。"
- 仲景风格：术语密度高，无连接词，直接堆叠

**仿写提示**：
- 写作前先列出本段涉及的术语
- 让术语自然堆叠，少用"因为……所以……"
- 术语之间靠"病机逻辑"串接

### 维度 6：经验性（Empiricism）

**特征**：
- 不讲"为什么"（理论）
- 只讲"是什么"（现象）+ "怎么办"（治法）
- 是临床经验的高度凝练

**典型**：
- "**呕家有痈脓者，不可治呕，脓尽自愈**。"—— 不解释为什么
- "**病者素不应食，而反暴思之，必发热也**。"—— 不解释机理

**对比**：
- 理论派："此因胃中郁热，热扰神明，故病人烦躁不欲饮食……"（理论解释）
- 仲景派："病人烦躁不欲饮食。"（现象）+ "调胃承气汤主之。"（治法）

**仿写提示**：
- 不写"因为 X，所以 Y"
- 写"X 现象" + "对应方/治法"
- 让读者自己体会因果

## 四、术语体系（90%+ 完整度）

仲景的术语体系是**张仲景视角写作**的"词汇表"。以下分类列出核心术语，
并标注每个术语的**含义、典型出处、辨析要点**。

### 4.1 六经病名

| 术语 | 含义 | 典型出处 | 辨析要点 |
|------|------|----------|----------|
| 太阳病 | 病位在表，阳证初起 | "太阳之为病，脉浮，头项强痛而恶寒。" | 不等于"感冒"；是"阳证初起"的总称 |
| 阳明病 | 病位在里，阳热亢盛 | "阳明之为病，胃家实是也。" | "胃家"指胃肠系统；"实"指有形实邪 |
| 少阳病 | 病位在半表半里 | "少阳之为病，口苦，咽干，目眩也。" | 三个症状是提纲；非"少阳经"病 |
| 太阴病 | 病位在里，阴寒初盛 | "太阴之为病，腹满而吐，食不下，自利益甚，时腹自痛。" | 主要是脾阳虚 |
| 少阴病 | 病位在心肾，阳气虚衰 | "少阴之为病，脉微细，但欲寐也。" | 不是"少阳"加重版；是质的转变 |
| 厥阴病 | 病位在肝，阴阳错杂 | "厥阴之为病，消渴，气上撞心，心中疼热，饥而不欲食，食则吐蛔。" | 最复杂的一经；寒热错杂 |

**辨析关键**：
- 六经**不是经络**——是六个病位阶段
- 六经**不是六个独立病**——是疾病动态演变
- "传"和"变"：传 = 顺序发展；变 = 不规则变化

### 4.2 病机术语

| 术语 | 含义 | 典型出处 | 辨析要点 |
|------|------|----------|----------|
| 表 | 病在外、在皮肤腠理 | "脉浮者，病在表，可发汗。" | 表证 = 可汗；非表证不可汗 |
| 里 | 病在内、在脏腑 | "病在里，不可发汗。" | 里证 = 可下/可温；非里证不可下 |
| 半表半里 | 病在表里之间 | "少阳病，汗吐下三禁。" | 三禁：禁汗、禁吐、禁下 |
| 寒 | 病性属寒 | "脉沉紧者，属少阴，发热者当灸之。" | 寒则温之 |
| 热 | 病性属热 | "热者，白虎汤主之。" | 热则清之 |
| 虚 | 正气不足 | "脉微细，但欲寐者，属少阴。" | 虚则补之 |
| 实 | 邪气有余 | "胃家实者，调胃承气汤主之。" | 实则泻之 |
| 阴 | 病性属阴 | "脉沉微者，属少阴。" | 阴证多寒、多虚 |
| 阳 | 病性属阳 | "脉浮大者，属太阳。" | 阳证多热、多实 |
| 阳虚 | 阳气不足 | "太阳病，发汗，汗漏不止，其人恶风，小便难，四肢微急，难以屈伸者，桂枝加附子汤主之。" | 阳虚则外寒 |
| 阴虚 | 阴液不足 | "少阴病，下利六七日，咳而呕渴，心烦不得眠者，猪苓汤主之。" | 阴虚则内热 |
| 阳盛 | 阳热过亢 | "阳明病，但头汗出，身无汗，剂颈而还，小便不利，渴引水浆者，此为瘀热在里，身必发黄，茵陈蒿汤主之。" | 阳盛则热 |
| 阴盛 | 阴寒内盛 | "少阴病，脉沉者，急温之，宜四逆汤。" | 阴盛则寒 |

### 4.3 症状术语（高频）

| 术语 | 含义 | 典型出处 | 辨析要点 |
|------|------|----------|----------|
| 发热 | 体温升高或自觉发热 | "太阳病，发热，汗出，恶风，脉缓者，名为中风。" | 分"翕翕发热"、"蒸蒸发热"、"潮热"等 |
| 恶寒 | 怕冷 | "太阳病，或已发热，或未发热，必恶寒。" | 必见；与发热并见 |
| 恶风 | 怕风 | "太阳病，发热，汗出，恶风者，桂枝汤主之。" | 比恶寒轻；多汗时见 |
| 汗出 | 有汗 | "太阳中风，阳浮而阴弱。阳浮者，热自发；阴弱者，汗自出。" | 区分自汗、盗汗、战汗 |
| 无汗 | 没有汗 | "太阳病，头痛，发热，身疼，腰痛，骨节疼痛，恶风，无汗而喘者，麻黄汤主之。" | 表实证据 |

| 动 | 脉形如豆 | "阳明病，脉浮而紧者，必潮热，发作有时。但浮者，必盗汗出。" | 主痛、主惊 |

**脉诊辨析**：
- 仲景脉诊**不单用**——必须与症状合参
- 同一种脉可见于不同病（如浮脉：表证、虚证、热证均有）
- 同一种病可见不同脉（如太阳病：浮缓、浮紧、浮数均可见）
- **脉证合参**是仲景诊法的核心

**核心经方（必知）**：
- 桂枝汤、麻黄汤、大青龙汤、小青龙汤

**方名规律**：
- 一字方：散（侯氏黑散）、汤（甘草汤）

**剂型**：
- 汤：最常用；急症、重症
- 散：粉末；急用、和中
- 丸：缓用；慢病
- 蜜丸：补益剂常用
- 煎：药汁

## 五、关键辨析（原文 vs 后世传抄）

> 这是仿写工具最重要的"安全栏"。仲景原文经过千年传抄，存在大量**后世掺入**的内容。
> 仿写时必须清楚哪些是仲景的语气、哪些是后世的注解语气。

### 5.1 误传风险等级

| 等级 | 类型 | 占比 | 处理建议 |
|------|------|------|----------|
| 高危 | 不可用 | 约 5% | 完全不引用 |
| 中危 | 需谨慎 | 约 10% | 引用时需说明 |
| 低危 | 大致可信 | 约 80% | 可直接引用 |
| 极低危 | 多版本一致 | 约 5% | 可放心引用 |

### 5.2 典型辨析点

#### 辨析 1：序例是否仲景原文？

**问题**：
《伤寒论》现行本首篇"序例"——"夫伤寒论，盖祖述大圣人之意……"等大段文字，
后世考证为**王叔和整理时掺入**，非仲景手笔。

**证据**：
- 序例多用骈文、语气近儒
- 与正文（条文）风格迥异
- 与仲景序中自述"勤求古训，博采众方"语气矛盾

**辨析**：
- 仲景原文：**简洁、临床、术语化**
- 王叔和序例：**骈文、说理、教化**

**仿写提示**：
- 不模仿序例的骈文风格
- 模仿条文的简洁风格

#### 辨析 2：原文 vs 后世衍文

**问题**：
"桂枝汤方"条下的"上五味，㕮咀三味……"中的"㕮咀"二字，
在唐宋以后的版本中被改写为"切"或"锉"。

**辨析**：
- 㕮咀 = 咬碎（仲景时代用法）
- 切 = 切碎（唐宋用法）
- 锉 = 锉碎（明清用法）

**仿写提示**：
- 用"㕮咀"显示对仲景的考据深度
- 用"切"显得通俗
- 现代仿写可用"切"或"碎"

#### 辨析 3：方名演变

**问题**：
仲景原方名与现代方名常有出入。

| 仲景原名 | 后世通用名 | 辨析 |
|----------|------------|------|
| 桂枝汤 | 同 | 无变化 |
| 麻黄汤 | 同 | 无变化 |
| 葛根汤 | 同 | 无变化 |
| 小柴胡汤 | 同 | 无变化 |
| 大承气汤 | 同 | 无变化 |
| 理中汤 | 同 | 有丸剂型 |
| 炙甘草汤 | 复脉汤 | 后世名更常用 |
| 芍药甘草汤 | 同 | 无变化 |
| 麻黄杏仁甘草石膏汤 | 麻杏甘石汤 | 后世简写 |
| 桂枝加葛根汤 | 同 | 无变化 |
| 柴胡加龙骨牡蛎汤 | 同 | 无变化 |

**仿写提示**：
- 仲景原名更"古典"
- 现代通用名更"易读"
- 仿写时用仲景原名更地道

#### 辨析 4：剂量单位

**问题**：
仲景时代的"两、斤、升、合"与后世不同。

| 仲景剂量 | 现代换算 | 辨析 |
|----------|----------|------|
| 一两 | 约 15.6 克 | 汉代一两约 15.6g |
| 一斤 | 约 250 克 | 16 两 = 1 斤 |
| 一升 | 约 200 毫升 | 液体量 |
| 一合 | 约 20 毫升 | 1 升 = 10 合 |
| 一尺 | 约 23 厘米 | 汉代一尺 |
| 一方寸匕 | 约 2-3 克 | 药粉量 |
| 一钱匕 | 约 1.5-2 克 | 药粉量 |
| 一撮 | 约 1 克 | 约 4 方寸匕 |

**辨析**：
- 仲景原方剂量**很大**（一两约 15g）—— 是今日常用剂量的 2-3 倍
- 宋代以后剂量**逐渐减小**
- 明清以后多用"钱"—— 约 3-4g

**仿写提示**：
- 引用原方时，**保留原剂量**显示考据
- 注明现代换算便于理解
- 不要混用

#### 辨析 5："主之"vs"宜"vs"可与"

**问题**：
三者在仲景用法中有微妙差别。

| 用词 | 语气 | 场景 |
|------|------|------|
| 主之 | **专断**（必用此方） | 病机明确、方证对应 |
| 宜 | **商量**（推荐此方） | 病机不明确、有选择余地 |
| 可与 | **许可**（可以此方） | 试探性给药 |

**仿写提示**：
- 病机明确 → "主之"
- 病机不明 → "宜"
- 试探性 → "可与"
- 不用错位

#### 辨析 6：脉象描述的"主观性"

**问题**：
仲景描述脉象用词主观，与现代脉诊术语有差异。

| 仲景用词 | 现代术语 | 辨析 |
|----------|----------|------|
| 浮 | 浮脉 | 一致 |
| 沉 | 沉脉 | 一致 |
| 迟 | 迟脉 | 一致 |
| 数 | 数脉 | 一致 |
| 紧 | 紧脉 | 一致 |
| 弦 | 弦脉 | 一致 |
| 缓 | 缓脉 | 一致 |
| 滑 | 滑脉 | 一致 |
| 涩 | 涩脉 | 一致 |
| 微 | 微脉 | 一致 |
| 细 | 细脉 | 一致 |
| 弱 | 弱脉 | 一致 |
| 大 | 大脉 | 一致 |
| 动 | 动脉 | 一致 |
| 促 | 促脉 | 一致 |
| 结 | 结脉 | 一致 |
| 代 | 代脉 | 一致 |
| 芤 | 芤脉 | 一致 |
| 革 | 革脉 | 一致 |
| 伏 | 伏脉 | 一致 |

**辨析**：
- 脉名基本一致
- 但仲景脉象常与**症状合用**——单纯的脉象意义不大
- 仲景："脉浮紧，无汗，发热"—— 脉+症合一
- 后世："脉浮紧"—— 单纯脉象

**仿写提示**：
- 写脉象时**必须带症状**
- 不写"脉浮紧"—— 写"脉浮紧，无汗发热"
- 体现仲景"脉证合参"特点

#### 辨析 7：症状描述的层次

**问题**：
仲景症状描述分**主症、兼症、变症**三个层次。

| 层次 | 含义 | 典型句式 |
|------|------|----------|
| 主症 | 必有症状 | "太阳病，头痛，发热，汗出，恶风" |
| 兼症 | 或有症状 | "或渴，或腹中痛，或胁下痞硬" |
| 变症 | 转化后症状 | "医反下之，利遂不止，脉促者" |

**仿写提示**：
- 主症必须列全
- 兼症用"或"引出
- 变症用"若"、"反"引出

#### 辨析 8：时态与时间词

**问题**：
仲景时态系统与现代汉语不同。

| 仲景用法 | 含义 | 现代对应 |
|----------|------|----------|
| 不解 | 病未愈 | "未愈"、"仍存在" |
| 已 | 已经 | "已"、"已发生" |
| 始 | 刚 | "刚刚"、"开始" |
| 复 | 又 | "又"、"再次" |
| 更 | 又 | "再"、"改变" |
| 仍 | 仍然 | "仍然" |
| 反 | 反而 | "反而" |
| 即 | 立即 | "立即"、"就是" |
| 必 | 必定 | "必定"、"一定" |
| 当 | 应当 | "应当"、"将要" |
| 宜 | 适合 | "适合"、"应该" |

**仿写提示**：
- 用"不解"代替"未愈"
- 用"始"代替"刚刚"
- 用"复"代替"又"
- 用"反"代替"反而"
- 用"必"代替"必定"

#### 辨析 9：六经命名

**问题**：
"六经"是后世归纳的术语，仲景原文**不说"六经"**。

| 仲景原文 | 后世归纳 |
|----------|----------|
| "太阳病" | 太阳经病 |
| "阳明病" | 阳明经病 |
| "少阳病" | 少阳经病 |
| "太阴病" | 太阴经病 |
| "少阴病" | 少阴经病 |
| "厥阴病" | 厥阴经病 |

**辨析**：
- 仲景原文只说"X 阴病"、"X 阳病"
- "六经"是宋以后归纳
- 仲景"经"是另一层意思（"行其经尽"）

**仿写提示**：
- 用"太阳病"不用"太阳经"
- 用"阳明病"不用"阳明经"
- 保持仲景的简洁命名

#### 辨析 10：药物炮制

**问题**：
仲景原方中药物**不写炮制**——因为是默认操作。
后世版本常加"炙"、"炒"、"炮"等。

| 仲景原方 | 后世通用方 | 辨析 |
|----------|------------|------|
| 甘草 | 炙甘草 | 仲景默认用炙甘草（部分方） |
| 大黄 | 酒大黄/生大黄 | 仲景常用生大黄 |
| 附子 | 炮附子 | 仲景用炮附子 |
| 干姜 | 干姜 | 一致 |
| 半夏 | 姜半夏/法半夏 | 后世加炮制 |
| 麻黄 | 麻黄/炙麻黄 | 仲景用生麻黄 |

**仿写提示**：
- 用"甘草"还是"炙甘草"看具体方
- 桂枝汤用"桂枝、芍药、甘草、生姜、大枣"
- 炙甘草汤用"炙甘草"
- 不可混用

## 六、句式 vs 后世：风格对照

为便于仿写时识别"什么是仲景语气、什么是后世语气"，列出 10 组对照。

| 仲景原文风格 | 后世注解风格 |
|--------------|--------------|
| "太阳病，头痛，发热，汗出，恶风，桂枝汤主之。" | "此太阳中风证也，营卫不和，故头痛发热汗出恶风，当用桂枝汤调和营卫。" |
| "少阴病，脉沉者，急温之，宜四逆汤。" | "少阴病脉沉，乃阳气虚衰之象，须急用四逆汤回阳救逆。" |
| "病人脉微而涩者，此为医所病也。" | "此医家误治所致也，脉微为阳气虚，涩为阴血少，故云。" |
| "呕家有痈脓者，不可治呕，脓尽自愈。" | "呕家有痈脓者，痈脓未溃，毒邪上逆，故呕。不可止呕，待脓尽则呕自止。" |
| "病者素不应食，而反暴思之，必发热也。" | "病人素不思食，今反暴思，此胃热之象，热气熏蒸，故发热也。" |
| "发汗后，恶寒者，虚故也；不恶寒，但热者，实也。" | "发汗后恶寒，乃阳气外泄而虚；不恶寒但热，乃邪气入里而成实。" |
| "伤寒一日，太阳受之，脉若静者，为不传。" | "伤寒一日，太阳经受邪，若脉象静而不数者，此邪气未传，仍在太阳。" |
| "少阳病，欲解时，从寅至辰上。" | "少阳属木，应春生之气，寅至辰乃少阳旺时，故欲解也。" |
| "太阳中风，脉浮紧，发热恶寒，身疼痛，不汗出而烦躁者，大青龙汤主之。" | "此太阳伤寒兼内热烦躁也，脉浮紧为寒束于表，烦躁为热郁于里，故用大青龙汤外散风寒，内清郁热。" |
| "少阴病，下利止而头眩，时时自冒者，死。" | "少阴病下利止，乃阴液枯竭，头眩自冒，为阴阳离决之象，故主死。" |

**风格对照要点**：
- 仲景：**极简、判断、不解释**——一句话，一个判断
- 后世：**展开、说理、加形容词**——一段话，一个解释
- 仿写时**只用前者**，不用后者

## 七、写作语法速查

仿写张仲景视角时，可以直接套用的"句法清单"。

### 7.1 开头句式

```
1. 病人[主症]，[主症]，[主症]，[方名]主之。
2. 太阳病，[主症]，[主症]，[主症]，[方名]主之。
3. [病名]之为病，[主症]也。
4. 伤寒[日数]日，[主症]，[主症]者，[方名]主之。
5. 病[主症]，[主症]者，[病机名]也。
6. 病人[主症]，[主症]者，属[经名]也。
```

### 7.2 病机推导句式

```
1. [症状群]者，[病机]也。
2. [病机 A]，[病机 B]，故[症状/结果]。
3. 夫[大前提]，[小前提]，是以[结论]。
4. [条件]，[推理]，[结论]，[方名]主之。
5. 病[主症]，[主症]，[主症]者，此[病机]也。
```

### 7.3 处方句式

```
1. [证候]者，[方名]主之。
2. [证候]，宜[方名]。
3. [证候]，可与[方名]。
4. [证候]，[方名]主之；[兼症]者，[方名加减]主之。
5. [证候]，[方名]主之。若[变症]，则[另一方名]主之。
```

### 7.4 预后句式

```
1. [证候]者，死。
2. [证候]者，难治。
3. [证候]者，不治。
4. [证候]者，可治。
5. [证候]，[方名]主之。服[药]已，[反应]，则愈。
```

### 7.5 禁忌句式

```
1. [证候]者，不可[治法]。
2. [病机]者，勿[操作]。
3. [误治]者，[后果]。
4. [证候]者，不可[治法]；[错误治法]则[后果]。
```

## 九、仿写使用指南

> 这一节是工具书——告诉仿写者如何使用本篇材料。

### 9.1 三步仿写法

**第一步：选句式**
- 写**判断性内容** → 用句式 1（"……者，……也"）
- 写**分类内容** → 用句式 2（"……者，……者"）
- 写**条件推进** → 用句式 3（"若……者"）
- 写**处方决定** → 用句式 4 或 5（"……宜……汤" 或 "……者，……汤主之"）
- 写**病机推导** → 用句式 6（"……故也"）
- 写**预后判断** → 用句式 7（"……者死"）
- 写**治疗禁忌** → 用句式 8（"……者，不可……"）

**第二步：套术语**
- 主症用 4.3 节
- 脉象用 4.4 节
- 治法用 4.5 节
- 方剂用 4.6 节
- 时间用 4.9 节
- 部位用 4.10 节
- 数字用 4.11 节
- 仲景特色用 4.12 节

**第三步：套风格**
- 简练（维度 1）
- 判断（维度 2）
- 客观（维度 3）
- 节律（维度 4）
- 术语密度（维度 5）
- 经验性（维度 6）

### 9.2 常见错误避免

| 错误类型 | 错误示例 | 正确示例 |
|----------|----------|----------|
| 加形容词 | "病人感觉非常寒冷" | "病人恶寒" |
| 解释机理 | "因为寒邪入侵，所以恶寒" | "恶寒" |
| 用"也许" | "这也许是少阴病" | "属少阴也" |
| 加连接词 | "由于……所以……" | 直接用"故" |
| 心理描写 | "病人担心自己的病" | 不写 |
| 反问句 | "这难道不是少阴病吗？" | "少阴病也" |
| 复合长句 | "病人经过发汗后，仍然不解" | "发汗后，不解" |
| 现代术语 | "病人出现呼吸衰竭" | "病人喘" |
| 西方概念 | "病人血压下降" | "病人脉微" |

### 9.3 长度建议

| 写作场景 | 句数 | 字数 |
|----------|------|------|
| 短引语 | 1-2 句 | 10-30 字 |
| 短判断 | 3-5 句 | 30-80 字 |
| 中等叙事 | 6-10 句 | 80-200 字 |
| 长段医案 | 11-20 句 | 200-400 字 |
| 完整诊治 | 20+ 句 | 400+ 字 |

**仲景风格：宁短勿长。** 一段话 80% 应该在 200 字以内。

### 9.4 节奏建议

- 4 字起步 + 2-3 字短句连珠 + 4 字收束
- 用顿号（、）连接同类项
- 一个"也"或"主之"结尾
- 避免长定语、长状语
- 避免复合从句

## 附录 A：仲景原文 30 例（按句式分类）

### A.1 句式 1："……者，……也"判断式（5 例）

1. 病人脉微而涩者，此为医所病也。
2. 太阳之为病，脉浮，头项强痛而恶寒。
3. 阳明之为病，胃家实是也。
4. 少阳之为病，口苦，咽干，目眩也。
5. 少阴之为病，脉微细，但欲寐也。

### A.2 句式 2："……者，……者"对举式（3 例）

1. 太阳病，发热，汗出，恶风，脉缓者，名为中风；太阳病，或已发热，或未发热，必恶寒，体痛，呕逆，脉阴阳俱紧者，名为伤寒。
2. 发汗后，恶寒者，虚故也；不恶寒，但热者，实也。
3. 病者身大热，反欲得衣者，热在皮肤，寒在骨髓也；病者身大寒，反不欲近衣者，寒在皮肤，热在骨髓也。

### A.3 句式 3："若……者"条件式（3 例）

1. 太阳病，若吐、若下、若温针，仍不解者，此为坏病。
2. 伤寒五六日，呕而发热者，柴胡汤证具。
3. 少阴病，下利止而头眩，时时自冒者，死。

### A.4 句式 4："……宜……汤"（3 例）

1. 太阳病三日，发汗不解，蒸蒸发热者，属胃也，调胃承气汤主之。
2. 少阴病，脉沉者，急温之，宜四逆汤。
3. 伤寒四五日，腹中痛，若转气下趣少腹者，此欲自利也。

### A.5 句式 5："……者，……汤主之"经典方证（10 例）

1. 太阳病，头痛，发热，汗出，恶风者，桂枝汤主之。
2. 太阳病，脉浮紧，无汗，发热，身疼痛，八九日不解，表证仍在，此当发其汗。
3. 太阳中风，脉浮紧，发热恶寒，身疼痛，不汗出而烦躁者，大青龙汤主之。
4. 太阳病，项背强几几，无汗恶风，葛根汤主之。
5. 太阳与阳明合病，喘而胸满者，不可下，宜麻黄汤。
6. 阳明病，潮热，大便微硬者，可与大承气汤。
7. 少阳病，脉弦细，头痛发热者，属少阳，不可发汗。
8. 伤寒五六日，中风，往来寒热，胸胁苦满，默默不欲饮食，心烦喜呕，小柴胡汤主之。
9. 少阴病，下利清谷，里寒外热，手足厥逆，脉微欲绝，通脉四逆汤主之。
10. 厥阴病，消渴，气上撞心，心中疼热，饥而不欲食，食则吐蛔。

### A.6 句式 6："……故也"病机推导（3 例）

1. 血弱气尽，腠理开，邪气因入，与正气相搏，结于胁下。
2. 病者素不应食，而反暴思之，必发热也。
3. 呕家有痈脓者，不可治呕，脓尽自愈。

### A.7 句式 7："……者死"预后（3 例）

1. 少阴病，下利止而头眩，时时自冒者，死。
2. 伤寒六七日，脉微，手足厥冷，烦躁，灸厥阴，厥不还者，死。
3. 病六七日，其人发热，暴利，汗出不止者，难治。

### A.8 句式 8："……者，不可……"禁忌（3 例）

1. 太阳病，桂枝证，医反下之，利遂不止，脉促者，表未解也。
2. 呕家有痈脓者，不可治呕，脓尽自愈。
3. 疮家，虽身疼痛，不可发汗。
